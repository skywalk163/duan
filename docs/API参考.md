## 语法元素

### 关键字

| 关键字 | 用途 | 示例 |
|--------|------|------|
| `定义` | 变量声明 | `定义 x 等于 10。` |
| `等于` | 赋值操作 | `x 等于 20。` |
| `如果` | 条件判断 | `如果 x > 10 那么:` |
| `那么` | 条件分支开始 | `如果 x > 10 那么:` |
| `否则` | 否则分支 | `否则:` |
| `否则如果` | 多条件分支 | `否则如果 x > 5 那么:` |
| `结束` | 块结束 | `结束。` |
| `当` | 循环 | `当 i < 10:` |
| `遍历` | 遍历循环 | `遍历 x 于 列表:` |
| `返回` | 返回值 | `返回 x。` |
| `跳出` | 跳出循环 | `跳出。` |
| `跳过` | 跳过当前迭代 | `跳过。` |
| `类` | 类定义 | `《人》类:` |
| `接口` | 接口定义 | `《可打印》接口:` |
| `方法` | 方法定义 | `《说话》方法():` |
| `段` | 段落定义 | `《加》段(a, b):` |
| `新` | 实例化 | `新 人()` |
| `之` | 属性/方法访问 | `对象之属性` |
| `导出` | 导出符号 | `导出 《加》。` |
| `导入` | 导入符号 | `导入 《加》。` |
| `从` | 从模块导入 | `从 math 导入 《加》。` |
| `真` | 布尔真值 | `定义 开关 等于 真。` |
| `假` | 布尔假值 | `定义 开关 等于 假。` |
| `空` | 空值 | `定义 x 等于 空。` |
| `打印` | 输出到控制台 | `打印("你好")。` |
| `输出` | 输出到控制台 | `输出("你好")。` |
| `尝试` | 异常处理 | `尝试:` |
| `捕获` | 捕获异常 | `捕获 错误:` |
| `抛出` | 抛出异常 | `抛出 错误。` |

### 标点符号

| 符号 | 用途 | 示例 |
|------|------|------|
| `《》` | 段落名/方法名 | `《加》段(a, b):` |
| `：` 或 `:` | 块开始 | `如果 x > 10 那么:` |
| `。` | 语句结束 | `定义 x 等于 10。` |
| `，` 或 `,` | 参数分隔 | `《加》(1, 2)` |
| `[]` | 列表 | `[1, 2, 3]` |
| `{}` 或 `_典()` | 字典 | `_典("名": "值")` |

## 控制流程

### 条件语句

```段言
如果 条件 那么:
    # 条件为真时执行
否则如果 其他条件 那么:
    # 其他条件为真时执行
否则:
    # 所有条件都不满足时执行
结束。
```

### 循环语句

```段言
# 当循环
当 条件:
    # 循环体
    跳出。  # 可选：跳出循环
    跳过。  # 可选：跳过当前迭代
结束。

# 遍历循环
遍历 元素 于 列表:
    打印(元素)。
结束。
```

## 段落（函数）

### 定义段落

```段言
《段落名》段(参数1, 参数2):
    # 段落体
    返回 结果。
结束。
```

### 调用段落

```段言
定义 结果 等于 《段落名》(参数1, 参数2)。
```

### 递归调用

```段言
《阶乘》段(n):
    如果 n 小于等于 1 那么:
        返回 1。
    结束。
    返回 n 乘 《阶乘》(n 减 1)。
结束。
```

## 类和对象

### 定义类

```段言
《类名》类:
    定义 属性1 等于 默认值1。
    定义 属性2 等于 默认值2。
    
    《初始化》方法(参数1, 参数2):
        属性1 等于 参数1。
        属性2 等于 参数2。
    结束。
    
    《方法名》方法():
        # 方法体
    结束。
结束。
```

### 实例化对象

```段言
定义 对象 等于 新 类名(参数1, 参数2)。
```

### 访问属性和方法

```段言
# 访问属性
定义 值 等于 对象之属性1。

# 调用方法
对象之方法名()。
```

### 继承

```段言
《子类》类 继承 父类:
    # 子类定义
结束。
```

### 导出符号

```段言

```

## 标准库

### math_utils 模块

```段言
从 math_utils 导入 《求和》, 《平均值》。

定义 列表 等于 [1, 2, 3, 4, 5]。
定义 总和 等于 《求和》(列表)。    # 15
定义 平均 等于 《平均值》(列表)。   # 3.0
```

### string_utils 模块

```段言
从 string_utils 导入 《反转串》, 《首字母大写》。

定义 原串 等于 "hello"。
定义 反转 等于 《反转串》(原串)。       # "olleh"
定义 大写 等于 《首字母大写》(原串)。   # "Hello"
```

### data_models 模块

```段言
从 data_models 导入 点, 矩形。

定义 p 等于 新 点(3, 4)。
定义 距离 等于 p之距离(新 点(0, 0))。  # 5.0

定义 r 等于 新 矩形(0, 0, 10, 5)。
定义 面积 等于 r之面积()。             # 50
```
