# 标准库

## 内置函数

### 列表函数

| 函数 | 说明 | 示例 |
|------|------|------|
| `长度` | 获取列表长度 | `长度 列表` |
| `追加` | 在列表末尾添加元素 | `列表 追加 元素` |
| `弹出` | 弹出并返回最后一个元素 | `列表 弹出` |
| `排序` | 对列表原地排序 | `列表 排序` |
| `反转` | 反转列表 | `列表 反转` |
| `包含` | 检查是否包含元素 | `列表 包含 元素` |

### 字符串函数

| 函数 | 说明 | 示例 |
|------|------|------|
| `长度` | 获取字符串长度 | `长度 字符串` |
| `替换` | 替换子串 | `字符串 替换 "旧" "新"` |
| `分割` | 分割字符串 | `字符串 分割 ","` |
| `去空格` | 去除首尾空格 | `字符串 去空格` |
| `子串` | 截取子串 | `字符串 子串 0 5` |

### 数学函数

| 函数 | 说明 | 示例 |
|------|------|------|
| `绝对值` | 绝对值 | `绝对值 -5` |
| `最大值` | 最大值 | `最大值 1 2 3` |
| `最小值` | 最小值 | `最小值 1 2 3` |
| `求和` | 求和 | `求和 列表` |
| `随机数` | 生成随机数 | `随机数 1 100` |
