# 段言（Duan）编程语言 - 语法扩展设计

## 概述

本文档定义段言编程语言的语法扩展，包括：
- 类与对象
- 模块系统
- 异常处理

设计原则：
1. 保持中文语法风格
2. 简洁易读
3. 与现有语法一致

## 1. 类与对象

### 1.1 类定义

**语法：**
```
类 类名：
  属性列表
  
  构造(参数)：
    初始化代码
  
  方法名(参数)：
    方法代码
```

**示例：**
```段言
类 学生：
  属性 姓名
  属性 年龄
  
  构造(姓名, 年龄)：
    定义己姓名等于姓名。
    定义己年龄等于年龄。
  
  介绍：
    打印\"我叫\"加己姓名加\"，今年\"加己年龄加\"岁。\"。
```

**生成 Python：**
```python
class 学生:
    def __init__(self, 姓名, 年龄):
        self.姓名 = 姓名
        self.年龄 = 年龄
    
    def 介绍(self):
        print(f"我叫{self.姓名}，今年{self.年龄}岁。")
```

### 1.2 继承

**语法：**
```
类 子类 继承 父类：
  方法重写
```

**示例：**
```段言
类 大学生 继承 学生：
  属性 专业
  
  构造(姓名, 年龄, 专业)：
    父类构造(姓名, 年龄)。
    定义己专业等于专业。
  
  介绍：
    打印\"我叫\"加己姓名加\"，今年\"加己年龄加\"岁，专业是\"加己专业加\"。\"。
```

### 1.3 属性访问

**语法：**
- `己属性` - self.属性
- `对象.属性` - 对象.属性

**示例：**
```段言
定义学生等于创建学生参数\"张三\"，20。
学生.介绍。
定义年龄等于学生.年龄。
```

## 2. 模块系统

### 2.1 模块定义

**语法：**
```
模块 模块名：
  导出内容
```

**示例：**

**文件：math.duan**
```段言
模块 数学工具：
  导出加法
  导出乘法
  
  段 加法(甲, 乙)：
    返回甲加乙。
  
  段 乘法(甲, 乙)：
    返回甲乘乙。
```

### 2.2 模块导入

**语法：**
```
从 模块名 导入 函数名
从 模块名 导入 函数1, 函数2
导入 模块名
```

**示例：**
```段言
从 数学工具 导入 加法, 乘法。

定义结果等于加法参数三，五。
打印结果。
```

**生成 Python：**
```python
from math_tool import add, multiply

result = add(3, 5)
print(result)
```

### 2.3 标准库导入

**语法：**
```
从 标准库 导入 模块名
```

**示例：**
```段言
从 标准库 导入 json。
从 标准库 导入 os。

定义数据等于json.加载参数\"data.json\"。
```

## 3. 异常处理

### 3.1 基本异常处理

**语法：**
```
尝试：
  可能出错的代码
捕获 错误类型 为 错误变量：
  错误处理代码
```

**示例：**
```段言
尝试：
  定义文件等于打开参数\"data.txt\"。
  定义内容等于文件.读取。
  打印内容。
捕获 文件错误 为 错误：
  打印\"无法读取文件：\"加错误.信息。
```

**生成 Python：**
```python
try:
    file = open("data.txt")
    content = file.read()
    print(content)
except FileNotFoundError as error:
    print(f"无法读取文件：{error}")
```

### 3.2 多异常捕获

**语法：**
```
尝试：
  代码
捕获 错误类型1 为 错误1：
  处理1
捕获 错误类型2 为 错误2：
  处理2
最终：
  清理代码
```

**示例：**
```段言
尝试：
  定义数值等于转换整数参数\"abc\"。
捕获 数值错误 为 错误：
  打印\"不是有效的数字\"。
捕获 其他错误 为 错误：
  打印\"发生未知错误\"。
最终：
  打印\"处理完成\"。
```

### 3.3 抛出异常

**语法：**
```
抛出 错误类型 参数 \"错误信息\"
```

**示例：**
```段言
段 检查年龄(年龄)：
  如果年龄小于零：
    抛出 数值错误 参数 \"年龄不能为负数\"。
  返回年龄。
```

### 3.4 自定义异常

**语法：**
```
异常 异常名 继承 基类异常：
  属性定义
```

**示例：**
```段言
异常 年龄错误 继承 异常：
  构造(年龄)：
    定义己信息等于\"无效的年龄：\"加年龄。
```

## 4. 其他扩展

### 4.1 装饰器

**语法：**
```
装饰器 装饰器名：
  装饰器代码

@装饰器名
段 函数名：
  函数代码
```

**示例：**
```段言
装饰器 计时：
  段 包装(函数)：
    段 执行(参数)：
      定义开始等于时间.当前。
      定义结果等于函数参数参数。
      定义结束等于时间.当前。
      打印\"执行时间：\"加(结束减开始)。
      返回结果。
    返回执行。

@计时
段 计算斐波那契(数)：
  返回斐波那契参数数。
```

### 4.2 上下文管理器

**语法：**
```
使用 资源 为 变量名：
  使用资源的代码
```

**示例：**
```段言
使用 打开参数\"data.txt\" 为 文件：
  定义内容等于文件.读取。
  打印内容。
```

**生成 Python：**
```python
with open("data.txt") as file:
    content = file.read()
    print(content)
```

### 4.3 列表推导式

**语法：**
```
[表达式 遍历 变量 在 列表 如果 条件]
```

**示例：**
```段言
定义列表等于1，2，3，4，5。
定义平方等于[数字乘数字 遍历 数字 在 列表 如果 数字大于二]。
打印平方。  # 输出：[9, 16, 25]
```

### 4.4 字典推导式

**语法：**
```
{键表达式: 值表达式 遍历 变量 在 列表}
```

**示例：**
```段言
定义列表等于\"甲\"，\"乙\"，\"丙\"。
定义映射等于{字符:字符的长度 遍历 字符 在 列表}。
打印映射。  # 输出：{'甲': 1, '乙': 1, '丙': 1}
```

### 4.5 Lambda 表达式

**语法：**
```
匿名(参数): 表达式
```

**示例：**
```段言
定义平方等于匿名(数): 数乘数。
打印平方参数5。  # 输出：25

定义列表等于1，2，3，4，5。
定义筛选等于列表.筛选参数匿名(数): 数大于二。
打印筛选。  # 输出：[3, 4, 5]
```

## 5. 语法关键字总结

### 5.1 新增关键字

**类相关：**
- `类` - 类定义
- `继承` - 继承关系
- `属性` - 类属性
- `构造` - 构造函数
- `己` - self 引用

**模块相关：**
- `模块` - 模块定义
- `导入` - 导入模块
- `从` - 导入来源
- `导出` - 导出符号
- `标准库` - 标准库标识

**异常相关：**
- `尝试` - try
- `捕获` - except
- `抛出` - raise
- `最终` - finally
- `异常` - exception 定义

**其他：**
- `装饰器` - decorator
- `使用` - with
- `匿名` - lambda

### 5.2 关键字优先级

保持双字关键字优先原则：
- 所有核心关键字统一为双字
- 单字保留给特殊用途（如 `己` 表示 self）

## 7. 示例程序

### 7.1 完整类定义

```段言
类 银行账户：
  属性 账户号
  属性 余额
  
  构造(账户号, 初始余额)：
    定义己账户号等于账户号。
    定义己余额等于初始余额。
  
  存款(金额)：
    如果金额小于等于零：
      抛出 数值错误 参数 \"存款金额必须大于零\"。
    定义己余额等于己余额加金额。
    打印\"存款成功，当前余额：\"加己余额。
  
  取款(金额)：
    如果金额小于等于零：
      抛出 数值错误 参数 \"取款金额必须大于零\"。
    如果金额大于己余额：
      抛出 数值错误 参数 \"余额不足\"。
    定义己余额等于己余额减金额。
    打印\"取款成功，当前余额：\"加己余额。
  
  查询余额：
    返回己余额。

定义账户等于创建银行账户参数\"12345\"，1000。
账户.存款参数500。
账户.取款参数200。
打印账户.查询余额。
```

捕获 文件错误 为 错误：
  打印\"文件错误：\"加错误.信息。
捕获 格式错误 为 错误：
  打印\"格式错误：\"加错误.信息。
