# 段言（Duan）使用指南

## 安装

### 环境要求

- Python 3.10+
- pip（Python 包管理器）

### 安装依赖

```bash
# 克隆项目
git clone https://github.com/dumate/duan.git
cd duan

# 安装 ANTLR 运行时
pip install antlr4-python3-runtime

# 安装测试依赖（可选）
pip install -r requirements-dev.txt
```

---

## 编写段言程序

### 文件格式

段言源文件使用 `.duan` 扩展名（或 `.段`）。

```段言
# hello.duan
打印 "你好，世界！"。
```

### 注释

```段言
# 这是单行注释（井号开头）

// 这也是单行注释（双斜杠开头）

注 这还是单行注释（中文"注"开头）
```

### 语句结束

每个语句以中文句号 `。` 结束：

```段言
设 甲 为 10。
设 乙 为 20。
```

## 运行段言程序

### 方法一：使用 CLI 编译器

```bash
# 编译并运行
python cli/duanc.py examples/hello.duan --run

# 仅编译（输出 Python 代码）
python cli/duanc.py examples/hello.duan

# 指定输出文件
python cli/duanc.py examples/hello.duan -o output.py
```

### 方法二：使用 Python 后端

```python
from src.duan_parser_v3 import DuanParser
from src.code_generator import PythonCodeGenerator

# 读取源文件
with open('examples/hello.duan', 'r', encoding='utf-8') as f:
    source = f.read()

# 解析
parser = DuanParser()
module = parser.parse(source)

# 生成 Python 代码
generator = PythonCodeGenerator()
python_code = generator.generate(module)

# 执行
exec(python_code)
```

### 方法三：使用 ANTLR CLI

```bash
# 运行段言脚本
python antlrparser/duan_cli.py run examples/hello.duan

# 显示 AST
python antlrparser/duan_cli.py parse examples/hello.duan

# 编译为可执行文件（LLVM）
python antlrparser/duan_cli.py compile examples/hello.duan -o output.exe
```

### 方法四：使用 REPL 交互环境

```bash
# 启动 REPL
python -m src.repl.core

# 或使用 ANTLR 版本
python antlrparser/duan_repl.py
```

REPL 支持的命令：

| 命令 | 功能 |
|------|------|
| `:help` | 显示帮助 |
| `:exit` 或 Ctrl+D | 退出 |
| `:clear` | 清屏 |
| `:reset` | 重置环境 |
| `:vars` | 显示变量 |
| `:tokens 段言代码` | 显示分词结果 |
| `:ast 段言代码` | 显示 AST |

## 完整示例

### 计算器

```段言
# calculator.duan - 简单计算器

段落 阶乘 接收 数值：
  如果 数值 小于等于 1：
    返回 1。
  结束。
  返回 数值 乘 阶乘(数值 减 1)。
结束。

打印 "5 的阶乘是："。
打印 阶乘(5)。
```

### 学生管理系统

```段言
# student_management.duan

类 学生：
  属性 姓名。
  属性 年龄。
  属性 成绩。
  
  构造 接收 姓名, 年龄, 成绩：
    己姓名 为 姓名。
    己年龄 为 年龄。
    己成绩 为 成绩。
  结束。
  
  段落 介绍：
    返回 "姓名：" 加 己姓名 加 "，年龄：" 加 己年龄 加 "，成绩：" 加 己成绩。
  结束。
结束。

设 学生列表 为 []。
设 学生1 为 新建 学生 "张三", 18, 95。
设 学生2 为 新建 学生 "李四", 19, 88。
列表追加 学生列表, 学生1。
列表追加 学生列表, 学生2。

遍历 学生项 之 学生列表：
  打印 学生项.介绍()。
结束。
```

## 常见问题

### Q: 中文变量名是否支持？
A: 完全支持。段言使用 UTF-8 编码，任何中文标识符都是合法的变量名。

### Q: 缩进要求是什么？
A: 段言使用缩进表示代码块。推荐使用 2 个空格缩进。`如果`、`当`、`遍历`、`段落`、`类` 等关键字后的代码块需要缩进。

### Q: 如何调试程序？
A: 可以使用 REPL 的 `:tokens` 和 `:ast` 命令查看分词和 AST 结果。也可以直接运行 Python 后端并查看生成的 Python 代码。

### Q: 支持哪些输出后端？
A: 目前支持 Python 后端（解释执行）和 LLVM 后端（编译为原生可执行文件）。

### Q: 遇到中文标点问题怎么办？
A: 段言支持全角标点，但注意语句结束必须使用中文句号 `。`，而非英文句号 `.`。

### Q: 如何理解编译器错误消息？
A: 段言提供结构化的错误提示，格式为 `错误类型 (行X, 列Y): 错误描述 (附近: '代码')`。例如：
```
词法错误 (行1, 列6): 未知字符: '￥' (0xFFE5)
语法错误 (行1,列4): 期望 '为'，得到 '三' (附近: '三')
语义错误: 变量 '甲' 重复定义
代码生成错误: 未知语句类型 (节点类型: SomeStmt)
```

### Q: 性能如何？
A: 经过优化后，段言编译器的性能指标为：
- 词法分析：约 0.053 毫秒/次
- 语法解析：约 0.060 毫秒/次
- 完整编译（33 项测试）：约 0.127 秒

## 最佳实践

1. **使用 v1.6 语法**：推荐使用 `设...为...`、`段落...接收...` 等新版语法
2. **合理使用注释**：用 `#` 注释说明代码意图
3. **模块化组织**：将功能分组到不同的 `.duan` 文件中，使用 `导入`/`导出` 管理依赖
4. **先测试后运行**：使用 `test_comprehensive_v2.py` 验证环境正常
5. **善用 REPL**：快速验证代码片段时使用 REPL 环境
