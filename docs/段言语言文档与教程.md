## 1. 段言是什么

段言（Duan）是一门**中文编程语言**，核心理念是「**词 → 句 → 段 → 篇**」的层级组织：

| 层级 | 含义 | 对应概念 |
|------|------|---------|
| **词** | 基础构件 | 变量、操作符 |
| **句** | 完整的语句 | 表达式、赋值、控制语句 |
| **段** | 可复用的代码块 | 函数 |
| **篇** | 完整的程序 | 模块 |

### 段言的设计目标

- **降低编程门槛**：用中文自然语言书写代码
- **提高语义密度**：用更少的代码表达更多的含义
- **保持简洁**：语法规则少，学习曲线平缓

### 一分钟感受段言

```duan
定义甲等于10。
定义乙等于20。
如果甲大于乙那么：
  打印甲。
否则：
  打印乙。

《平方》段(数值):
  返回数值乘数值。

定义结果等于《平方》(5)。
打印结果。
```

## 2. 安装与快速开始

### 环境要求

- Python 3.8+
- ANTLR 4 运行时

### 安装步骤

```bash
# 1. 克隆项目
git clone <项目地址>
cd duan

# 2. （推荐）创建虚拟环境
python -m venv .venv
# Windows:
.venv\Scripts\activate
# macOS/Linux:
source .venv/bin/activate

# 3. 安装依赖
pip install antlr4-python3-runtime

### 验证安装

```bash
cd antlrparser
python -m pytest test/ -v
```

如果所有测试通过（62 个测试），说明环境就绪。

```bash

```bash
pip install -e .

```

你应该能看到输出：

```

```

## 3. 基础语法速成

### 3.1 注释

```duan
# 这是单行注释

```注释
这是多行注释
可以跨越多行
```
```

### 3.2 变量定义

使用 `定义...等于...` 来定义变量：

```duan
定义甲等于10。           # 整数
定义乙等于3.14。         # 浮点数
定义丙等于真。           # 布尔值（真/假）
定义丁等于空。           # 空值
定义戊等于"你好"。       # 字符串（双引号）
定义己等于'hello'。      # 字符串（单引号）
```

变量名可以是中文或英文标识符。

### 3.3 打印输出

```duan
打印("你好，世界")。
打印(42)。
输出(甲)。
```

### 3.4 语句结束

段言中，句号 `。` 或 `.` 表示语句结束：

```duan
定义甲等于10。
定义乙等于20.       # 也可以用英文句号
```

分号 `;` 可用于在一行写多个语句：

```duan
定义甲等于10；定义乙等于20；打印甲加乙。
```

### 3.5 符号双模式

段言支持中英文标点符号混用，完全等价：

| 符号 | 中文 | 英文 |
|------|------|------|
| 句号 | `。` | `.` |
| 冒号 | `：` | `:` |
| 逗号 | `，` | `,` |
| 分号 | `；` | `;` |
| 括号 | `（）` | `()` |

```duan
# 以下三种写法完全等价
《计算》段(甲: 数, 乙: 数):
  返回甲加乙。

《计算》段(甲: 数, 乙: 数)：
  返回甲加乙。

《计算》段（甲：数，乙：数）：
  返回甲加乙。
```

**推荐风格**：语义符号用中文（句号、冒号），辅助符号用英文（括号、逗号）。

## 4. 数据类型与运算

### 4.1 基本数据类型

| 类型 | 示例 | 说明 |
|------|------|------|
| **整数** | `10`、`-5` | 整数值 |
| **浮点数** | `3.14`、`-0.5` | 带小数点的数值 |
| **字符串** | `"你好"`、`'hello'` | 文本（支持双引号和单引号） |
| **布尔值** | `真`、`假` | 逻辑值 |
| **空值** | `空` | 表示无值 |
| **列表** | `【1, 2, 3】` | 有序集合 |

### 4.2 算术运算

| 运算 | 中文写法 | 符号写法 |
|------|---------|---------|
| 加法 | `甲加乙` | `甲 + 乙` |
| 减法 | `甲减乙` | `甲 - 乙` |
| 乘法 | `甲乘乙` | `甲 × 乙` |
| 除法 | `甲除乙` | `甲 ÷ 乙` |
| 取模 | `甲模乙` | `甲 % 乙` |
| 幂运算 | `甲幂乙` | `甲 ^ 乙` |

```duan
定义结果_1等于10加5。       # 15
定义结果_2等于10减5。       # 5
定义结果_3等于10乘5。       # 50
定义结果_4等于10除5。       # 2.0
定义结果_5等于10模3。       # 1
定义结果_6等于2幂3。        # 8
```

### 4.3 比较运算

| 运算 | 写法 |
|------|------|
| 等于 | `甲等于乙` |
| 不等于 | `甲不等于乙` |
| 大于 | `甲大于乙` |
| 小于 | `甲小于乙` |
| 大于等于 | `甲大于等于乙` |
| 小于等于 | `甲小于等于乙` |

```duan
定义_比较1等于10大于5。      # 真
定义_比较2等于10小于5。      # 假
定义_比较3等于10等于10。     # 真
```

### 4.4 逻辑运算

```duan
定义_逻辑1等于真且假。       # 假（AND）
定义_逻辑2等于真或假。       # 真（OR）
定义_逻辑3等于非真。         # 假（NOT）
```

### 4.5 运算符优先级

从高到低：

1. 括号 `()`
2. 幂 `^`
3. 乘、除、模 `× ÷ %`
4. 加、减 `+ -`
5. 比较 `> < >= <= == !=`
6. 非 `非`
7. 且 `且`
8. 或 `或`

```duan
定义结果等于(3加5)乘2。     # 16
```

### 4.6 字符串操作

```duan
# 字符串拼接
定义_拼接等于"hello"加"world"。     # "helloworld"

# 转义字符
定义_转义等于"第一行\n第二行"。       # 换行
定义_反斜杠等于"路径\\文件"。         # 反斜杠本身
```

### 4.7 列表

```duan
# 定义列表
定义_数字列等于【1, 2, 3, 4, 5】。
定义_混合列等于【1, "你好", 真, 空】。

# 列表索引（从 0 开始）
定义_第一个等于_数字列[0]。          # 1
定义_第二个等于_数字列[1]。          # 2

# 列表长度
定义_长度等于_数字列之长度。          # 5

# 嵌套列表
定义_矩阵等于【【1, 2】, 【3, 4】】。
定义_元素等于_矩阵[0][1]。           # 2
```

## 5. 控制流

### 5.1 条件判断（如果-那么-否则）

```duan
# 单分支
如果甲大于乙那么：
  打印(甲)。

# 双分支
如果分数大于等于60那么：
  打印('及格')。
否则：
  打印('不及格')。

# 多分支（否则若）
如果分数大于等于90那么：
  打印('优秀')。
否则若分数大于等于80那么：
  打印('良好')。
否则若分数大于等于60那么：
  打印('及格')。
否则：
  打印('不及格')。
```

### 5.2 遍历循环

```duan
# 遍历列表
定义列表等于【1, 2, 3】。
遍历 项 列表：
  打印(项)。

# 遍历字典
遍历 键，值 典：
  打印(键加"："加值)。
```

### 5.3 当循环

```duan
定义计数等于3。
当计数大于0：
  打印(计数)。
  定义计数等于计数减1。
```

### 5.4 跳出与跳过

```duan
# 跳出循环
遍历 项 列表：
  如果项大于10那么：
    跳出。
  打印(项)。

# 跳过当前项
遍历 项 列表：
  如果项等于5那么：
    跳过。
  打印(项)。
```

## 6. 段落（函数）定义

### 6.1 定义段落

```duan
《段落名》段(参数1, 参数2):
  # 段落体
  返回 结果。
```

示例：

```duan
《平方》段(数值):
  返回数值乘数值。

《双倍》段(数值):
  返回数值乘2。

《阶乘》段(数值):
  如果数值小于等于1那么:
    返回1。
  定义结果等于1。
  定义_计数器等于数值。
  当_计数器大于1:
    结果等于结果乘_计数器。
    _计数器等于_计数器减1。
  返回结果。
```

### 6.2 调用段落

```duan
# 带括号调用（推荐）
定义结果等于《平方》(5)。         # 25

# 无括号调用
定义结果等于《阶乘》5。            # 120
```

### 6.3 段落返回值

```duan
《计算》段(甲, 乙):
  返回甲加乙。

定义和等于《计算》(3, 5)。        # 8
```

### 6.4 管道式调用

```duan
# 使用 -> 操作符
数据 -> 《过滤》 -> 《排序》 -> 《输出》。

# 使用并操作符
返回《快速排序》(小列表) -> 等列表 -> 《快速排序》(大列表)。
```

## 7. 模块与导入导出

### 7.1 导出段落

在模块文件中使用 `导出` 声明对外公开的段落：

```duan
# mod_math.duan
导出《平方》。
导出《立方》。

《平方》段(数值):
  返回数值乘数值。

《立方》段(数值):
  返回数值乘数值乘数值。
```

### 7.2 导入段落

```duan
# 从模块导入指定段落
从 mod_math 导入《平方》，《立方》。

# 使用导入的段落
定义结果等于《平方》(5)。        # 25
定义结果2等于《立方》(3)。       # 27
```

### 7.3 搜索路径

解释器默认在当前目录和 `test/` 目录下搜索模块：

```python
from duan_interpreter import run_source

interp = run_source(
    '从 mod_math 导入《平方》。',
    search_paths=['test', '.']
)
```

### 7.4 模块间依赖

模块可以导入其他模块，系统会自动递归解析依赖：

```duan
# mod_advanced.duan
导出《高级计算》。

从 mod_math 导入《平方》，《立方》。

《高级计算》段(数值):
  返回《平方》(数值)加《立方》(数值)。
```

## 9. CLI 命令行工具

```

```bash

```bash

```

```bash

```

输出示例：

## 10. REPL 交互环境

REPL（Read-Eval-Print Loop）提供了一个交互式编程环境，适合快速实验段言语法。

### 10.1 启动

```bash
duan repl
```

你会看到欢迎信息：

```
段言 (Duan) v0.1.0 - 中文编程语言交互环境

可用命令:
  /help     显示帮助信息
  /exit     退出 REPL
  /clear    清屏
  /env      显示当前环境

示例:
  >>> 打印("你好，世界")。
  >>> 定义甲等于42。
  >>> 甲
```

### 10.2 基本使用

```duan
段言> 打印("你好，段言！")。
你好，段言！

段言> 定义甲等于42。
42

段言> 甲
42

段言> 定义乙等于甲乘2。
84

段言> 乙
84
```

### 10.3 多行输入

REPL 自动检测不完整的代码并等待更多输入：

```duan
段言> 《双倍》段(数值):
   ...   返回数值乘2。
   ... 结束。

段言> 定义结果等于《双倍》(5)。
10
```

检测规则：
- 以冒号结尾 → 需要更多输入（块定义）
- 括号未匹配 → 需要更多输入（`【、（、《` 缺少对应的 `】、）、》`）

### 10.4 REPL 命令

| 命令 | 功能 |
|------|------|
| `/help` | 显示快速参考 |
| `/exit` | 退出 REPL |
| `/clear` | 清屏 |
| `/env` | 显示当前已定义的变量和段落 |

### 10.5 表达式自动求值

在 REPL 中输入单独的表达式会自动求值并打印结果：

```duan
段言> 3加5乘2
16
```

### 10.6 历史记录

REPL 会自动保存历史记录到 `~/.duan_history` 文件（支持方向键上下浏览历史）。

### 示例 3：模块化编程

**mod_math.duan**（模块文件）：

```duan
导出《平方》。
导出《阶乘》。

《平方》段(数值):

```

### 示例 4：字典使用

```duan
# 创建字典
定义学生等于_典("姓名", "张三", "年龄", 18, "成绩", 95)。

# 字典操作

## 12. 常见问题

### Q1: 如何安装运行时依赖？

```bash
pip install antlr4-python3-runtime
```

### Q2: 如何处理中文编码问题？

确保源码文件以 UTF-8 编码保存。所有文本编辑器默认都支持 UTF-8。

### Q3: 段言支持哪些 Python 版本？

Python 3.8 及以上版本。

### Q4: 如何调试代码？

1. 使用 `duan parse` 查看 AST 结构
2. 使用 `duan tokenize` 查看词法分析结果
3. 使用 REPL 环境逐段测试代码
4. 使用 `-v` 选项显示详细错误信息

### Q5: 模块找不到怎么办？

确保：
1. 模块文件在正确的目录中
2. 使用了正确的模块名（不包含扩展名）
3. 搜索路径包含模块所在目录

### Q6: 如何退出 REPL？

- 输入 `/exit` 或 `/quit`
- 按 `Ctrl+C` 两次
- 按 `Ctrl+D`（EOF）

### Q7: 中英文标点可以混用吗？

可以。段言支持所有中英文标点的等价替换（句号、冒号、逗号、括号等），可以在同一文件中自由混用。

### Q8: 变量的命名规范？

变量名可以使用中文、英文、数字和下划线，但不能以数字开头。推荐使用有语义的中文名。
