# 段言（Duan）编程语言 - 统一语法规范 v1.5

## 设计原则

1. **母语优先**：语法符合中文表达习惯，读起来像自然语言
2. **简洁明了**：减少标点符号，避免不必要的复杂性
3. **择优而用**：综合两个实现的优点
4. **易于学习**：降低学习曲线，让新手快速上手

---

## 核心语法对比

### 1. 变量声明

**src语法**：`设 甲 为 三。`
**antlrparser语法**：`定义甲等于10。`

**统一选择**：`设 甲 为 三。`
- 理由：更符合中文表达习惯，"设...为..."是标准中文句式
- 类型注解：`设 甲 为 三 其型为数。`（可选）

### 2. 函数定义

**src语法**：
```段言
段落 平方 数值。
  返回 数值 乘 数值。
结束。
```

**antlrparser语法**：
```段言
《平方》段(数值)：
  返回数值乘数值。
```

**统一选择**：
```段言
段落 平方 接收 数值。
  返回 数值 乘 数值。
结束。
```
- 理由：保持src的段落关键字，"接收"明确参数含义
- 可选简写：`段落 平方。`（无参数）

### 3. 模块导入

**src语法**：`从《数学工具》导入 平方。`
**antlrparser语法**：`从 mod_math 导入《平方》，《阶乘》。`

**统一选择**：
```段言
导入《数学工具》。
从《数学工具》导入 平方 阶乘。
导入《模块名》为 别名。
```
- 理由：书名号《》符合中文习惯，明确模块边界

### 4. 类定义

**src语法**：
```段言
类 狗 继承 动物。
  属性 名称。
  属性 年龄。
  
  构造 接收 名称 年龄。
    己名称 为 名称。
    己年龄 为 年龄。
  结束。
  
  段落 说话。
    返回 "汪汪汪"。
  结束。
结束。
```

**antlrparser语法**：
```段言
《狗》类 继承 《动物》 使用 《可打印》:
  定义名称等于""。
  
  《狗》(name, age):
    《动物》(name)。
    年龄等于age。
  结束。
  
  《说话》方法 -> 串:
    返回"汪汪汪"。
  结束。
结束。
```

**统一选择**：
```段言
类 狗 继承 动物 实现 可打印。
  属性 名称 为 串。
  属性 年龄 为 数。
  
  构造 接收 名称 年龄。
    己名称 为 名称。
    己年龄 为 年龄。
  结束。
  
  段落 说话 返回 串。
    返回 "汪汪汪"。
  结束。
结束。
```
- 理由：保留src的"类...继承...实现"格式
- "属性...为..."明确属性类型
- "己"作为self引用，简洁明了

### 5. 条件语句

**src语法**：
```段言
如果 甲 大于 乙 那么：
  返回 甲。
否则：
  返回 乙。
结束。
```

**antlrparser语法**：
```段言
如果甲大于乙那么：
  打印甲。
否则：
  打印乙。
```

**统一选择**：
```段言
如果 甲 大于 乙：
  返回 甲。
否则：
  返回 乙。
结束。
```
- 理由：去掉"那么"，冒号足够明确
- 保留"结束"关键字，避免缩进歧义

### 6. 循环语句

**src语法**：
```段言
遍历 项目 于 列表：
  打印 项目。
结束。

当 条件：
  执行操作。
结束。
```

**antlrparser语法**：
```段言
遍历项目于列表：
  打印项目。
```

**统一选择**：保留src语法
```段言
遍历 项目 于 列表：
  打印 项目。
结束。

当 条件：
  执行操作。
结束。
```
- 理由："遍历...于..."符合中文表达习惯

### 7. 异常处理

**src语法**：
```段言
尝试：
  可能出错的代码。
捕获 异常变量：
  处理异常。
最终：
  清理代码。
结束。
```

**统一选择**：保留src语法
- 理由：清晰明确，符合主流语言习惯

## 标点符号使用规则

### 必须使用
1. **中文句号（。）**：语句结束符
2. **冒号（：）**：代码块开始
3. **书名号（《》）**：模块名、段落名引用

### 可选使用
1. **中文逗号（，）**：分隔多个参数或元素
2. **顿号（、）**：分隔同类型项

### 尽量避免
1. **括号（）**：函数调用和参数传递尽量用空格
2. **英文标点**：统一使用中文标点

### 示例对比

**不推荐**：
```段言
定义result等于《平方》(5)。
打印(result)。
```

**推荐**：
```段言
设 结果 为 平方 5。
打印 结果。
```

## 高级特性

### 1. 接口定义（借鉴antlrparser）

```段言
接口 可打印。
  段落 输出 返回 串。
结束。
```

### 2. 泛型支持（借鉴antlrparser）

```段言
类 列表 保存 类型。
  属性 数据 为 列。
  
  段落 添加 接收 项目 为 类型。
    数据 之 添加 项目。
  结束。
结束。
```

### 3. 类型别名

```段言
类型 用户名 为 串。
类型 年龄 为 数。
```

### 4. 数据类型定义

```段言
数据 用户。
  字段 姓名 为 串。
  字段 年龄 为 数。
结束。
```

```段言
# 导入标准库
导入《数学工具》。

# 定义一个类
类 狗 继承 动物。
  属性 名称 为 串。
  属性 年龄 为 数。

    己年龄 为 年龄。
  结束。
  
  段落 说话 返回 串。
    返回 "汪汪汪"。
  结束。
  
  段落 计算年龄。
    返回 平方 己年龄。
  结束。

# 主程序
段落 主程序。
  设 狗子 为 新建 狗 "旺财" 3。
  打印 狗子 之 名称。
  打印 狗子 之 说话。
结束。

主程序。
```

## 迁移指南

### 从src迁移
- 无需修改，src语法是统一规范的基础

### 从antlrparser迁移
- `定义甲等于10。` → `设 甲 为 十。`
- `《平方》段(数值)：` → `段落 平方 接收 数值。`
- `从 mod_math 导入《平方》` → `从《数学工具》导入 平方`
- `《狗》类 继承 《动物》:` → `类 狗 继承 动物。`
