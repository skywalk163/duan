# 段言编程语言 - 统一语法规范 v1.6（最终版）

**版本**：v1.6  
**日期**：2026-06-13  
**状态**：最终版

---

## 设计原则

1. **母语优先**：语法符合中文表达习惯，读起来像自然语言
2. **简洁明了**：减少标点符号，避免不必要的复杂性
3. **无空格分词**：支持连续书写，自动识别标识符和关键字边界
4. **元数驱动**：动词声明参数数量，自动收集参数，支持无括号调用
5. **择优而用**：综合两个实现（手写解析器 + ANTLR）的优点

## 一、词法规范

### 1.1 字符集

段言源文件使用 UTF-8 编码，支持：
- 中文字符（CJK 统一表意文字）
- 英文字母（a-z, A-Z）
- 数字（0-9）
- 中文标点符号

### 1.2 注释

```段言
# 井号注释
// 双斜杠注释
注 中文"注"开头注释
```

### 1.3 关键字

段言的关键字全部为**双字**（决策27）：

| 类别 | 关键字 |
|------|--------|
| 定义声明 | `定义`、`常量`、`类型`、`导入`、`导出`、`从`、`为`、`设` |
| 条件判断 | `如果`、`那么`、`否则`、`否则若` |
| 循环控制 | `遍历`、`当`、`跳出`、`跳过`、`在`、`之` |
| 段落定义 | `段`、`段落`、`参数`、`接收`、`返回`、`结束` |
| 异常处理 | `尝试`、`捕获`、`抛出`、`最终` |
| 类与对象 | `类`、`继承`、`属性`、`构造`、`新建` |
| 模块系统 | `模块`、`标准库` |
| 保留字 | `并`、`的`、`己` |

**单字保留字**：

| 字 | 含义 |
|----|------|
| `真` | True |
| `假` | False |
| `空` | None |
| `且` | 逻辑与 |
| `或` | 逻辑或 |
| `非` | 逻辑非 |
| `己` | self 引用 |

### 1.4 运算符关键字

| 类别 | 关键字 | 含义 |
|------|--------|------|
| 算术 | `加`、`减`、`乘`、`除`、`取余` | `+` `-` `*` `/` `%` |
| 复合算术 | `加上`、`减去`、`乘以`、`除以` | `+` `-` `*` `/` |
| 比较 | `大于`、`小于`、`等于`、`不等于`、`大于等于`、`小于等于` | `>` `<` `==` `!=` `>=` `<=` |

### 1.5 符号映射

| 中文符号 | 对应英文 | 用途 |
|---------|---------|------|
| `。` | `.` | 语句结束 |
| `，` | `,` | 参数分隔、管道操作 |
| `、` | `\` | 参数并列分隔 |
| `：` | `:` | 冒号（代码块开始） |
| `（` `）` | `(` `)` | 括号 |
| `【` `】` | `[` `]` | 方括号（索引访问） |
| `《` `》` | `<` `>` | 书名号（段落名、模块名） |
| `「` `」` | `"` | 双引号（字符串） |
| `『` `』` | `'` | 单引号（字符串） |

## 二、语法规范

### 2.1 变量声明

两种语法风格（新版推荐）：

```段言
# 风格一：设...为...（v1.6 推荐）
设 甲 为 10。
设 姓名 为 "张三"。
设 身高 为 1.75。

# 风格二：定义...等于...（v1.5 兼容）
定义 年龄 等于 25。
定义 已婚 等于 真。
```

### 2.2 赋值

```段言
甲 等于 20。       # 修改变量甲的值
己名称 为 "新名"。  # self 属性赋值（类方法中）
```

### 2.3 段落定义（函数）

```段言
# 风格一：段落...接收...（v1.6 推荐）
段落 平方 接收 数值：
  返回 数值 乘 数值。
结束。

段落 求和 接收 甲, 乙：
  返回 甲 加 乙。
结束。

段落 问候：
  打印 "你好，世界！"。
结束。

# 风格二：《名》段(参数)：（v1.5 兼容）
《加法》段(甲, 乙)：
  返回 甲 加 乙。
```

**参数说明**：
- 参数使用 `接收` 关键字声明
- 多个参数用逗号（`,`）分隔
- 参数列表后使用冒号（`：`），不使用句号
- 函数体使用 `结束。` 标记结束

### 2.4 函数调用

```段言
# 直接调用
打印 "你好"。
平方(5)。

# 变量接收返回值
设 结果 为 平方(5)。
设 和值 为 求和(10, 20)。
```

### 2.5 返回语句

```段言
段落 阶乘 接收 数值：
  如果 数值 小于等于 1：
    返回 1。
  结束。
  返回 数值 乘 阶乘(数值 减 1)。
结束。
```

### 2.6 条件语句

```段言
# 基本条件
如果 分数 大于等于 60：
  打印 "及格"。
结束。

# 带 else
如果 分数 大于等于 60：
  打印 "及格"。
否则：
  打印 "不及格"。
结束。

# 带 elif（否则若）
如果 分数 大于等于 90：
  打印 "优秀"。
否则若 分数 大于等于 80：
  打印 "良好"。
否则：
  打印 "加油"。
结束。
```

### 2.7 当循环

```段言
设 计数 为 0。
当 计数 小于 5：
  打印 计数。
  设 计数 为 计数 加 1。
结束。
```

### 2.8 遍历循环

```段言
设 水果 为 ["苹果", "香蕉", "橙子"]。
遍历 水果项 之 水果：
  打印 水果项。
结束。
```

### 2.9 跳出/跳过

```段言
当 计数 小于 10：
  如果 计数 等于 5：
    跳出。    # 跳出循环
  结束。
  如果 计数 等于 3：
    跳过。    # 跳过本次循环
  结束。
  打印 计数。
结束。
```

### 2.10 类定义

```段言
# 定义类
类 动物：
  段落 叫声：
    打印 "动物叫声"。
  结束。
结束。

# 类继承
类 狗 继承 动物：
  # 属性声明
  属性 名称。
  属性 品种。
  
  # 构造函数
  构造 接收 名称, 品种：
    己名称 为 名称。
    己品种 为 品种。
  结束。
  
  # 方法定义
  段落 叫声：
    打印 "汪汪汪"。
  结束。
  
  段落 介绍：
    打印 "我是"。
    打印 己名称。
  结束。
结束。
```

### 2.11 对象创建与使用

```段言
# 创建对象（新建 类名 参数...）
设 小狗 为 新建 狗 "旺财", "金毛"。

# 方法调用
小狗.叫声()。
小狗.介绍()。

# 属性访问
打印 小狗.名称。
```

### 2.12 模块系统

```段言
# 导入整个模块
导入 数学工具。

# 从模块导入特定符号
从 数学工具 导入 平方 阶乘。

# 带别名导入
导入 数学工具 为 数学。

# 导出符号
导出 平方 阶乘。
导出 全部。
```

### 2.13 异常处理

```段言
尝试：
  设 结果 为 除(10, 0)。
  打印 结果。
捕获 异常：
  打印 "发生错误："。
  打印 异常。
结束。
```

### 2.14 列表和字典

```段言
# 列表
设 列表 为 [1, 2, 3, 4, 5]。
设 首项 为 列表[0]。
设 长度 为 长(列表)。
列表追加 列表, 6。

# 字典
设 字典 为 {"姓名": "张三", "年龄": 25}。
设 姓名 为 字典["姓名"]。
```

### 2.15 管道操作

```段言
# 使用 -> 进行管道操作
数据 -> 过滤 -> 映射 -> 排序。
```

## 五、元数驱动解析

段言使用动词的元数（arity）自动收集参数（决策28）：

| 元数 | 含义 | 示例 |
|------|------|------|
| 0 | 无参数 | `参数列表` |
| 1 | 单参数 | `打印 甲`、`平方(5)` |
| 2 | 双参数 | `甲 加 乙`、`列表追加 列表, 项` |
| 3 | 三参数 | `替换字符串 源, 旧, 新` |
| -1 | 可变参数 | `列 1, 2, 3`、`新建 类 参数...` |
