# 段言编程语言 - 统一语法规范 v1.6（最终版）

## 设计原则

1. **母语优先**：语法符合中文表达习惯，读起来像自然语言
2. **简洁明了**：减少标点符号，避免不必要的复杂性
3. **择优而用**：综合两个实现的优点
4. **易于学习**：降低学习曲线，让新手快速上手

---

## 核心语法

### 1. 变量声明

**语法**：`设 名称 为 值。`

**示例**：
```段言
设 甲 为 十。
设 名字 为 "世界"。
设 结果 为 平方(5)。
```

**说明**：
- 使用"设...为..."句式，符合中文习惯
- 值可以是数字、字符串或表达式
- **注意**：语句以中文句号（。）结束

---

### 2. 段落定义（函数）

**语法**：
```段言
段落 名称 接收 参数：
  语句。
结束。
```

**示例**：
```段言
段落 平方 接收 数值：
  返回 数值 乘 数值。
结束。

段落 求和 接收 甲, 乙：
  返回 甲 加 乙。
结束。

段落 问候：
  打印 "你好，世界！"。
结束。
```

**说明**：
- 使用"段落"关键字定义函数
- 使用"接收"关键字声明参数
- 参数列表使用冒号（：）开始函数体
- 函数体使用"结束。"标记结束
- **重要**：参数列表后使用冒号，不是句号

---

### 3. 类定义

**语法**：
```段言
类 类名 继承 父类 实现 接口：
  属性 名称。
  
  构造 接收 参数：
    己.属性 为 参数。
  结束。
  
  段落 方法：
    语句。
  结束。
结束。
```

**示例**：
```段言
类 狗 继承 动物：
  属性 名称。
  属性 年龄。
  
  构造 接收 名称, 年龄：
    己.名称 为 名称。
    己.年龄 为 年龄。
  结束。
  
  段落 说话：
    返回 "汪汪汪"。
  结束。
结束。
```

**说明**：
- 使用"类"关键字定义类
- "继承"和"实现"关键字可选
- 使用"属性"关键字声明属性
- 使用"构造"关键字定义构造函数
- 使用"己"作为self引用

---

### 4. 控制流

#### if语句

**语法**：
```段言
如果 条件：
  语句。
否则：
  语句。
结束。
```

**示例**：
```段言
如果 甲 大于 乙：
  打印 甲。
否则：
  打印 乙。
结束。
```

#### 遍历循环

**语法**：
```段言
遍历 项目 于 列表：
  语句。
结束。
```

**示例**：
```段言
设 列表 为 [1, 2, 3, 4, 5]。
遍历 项目 于 列表：
  打印 项目。
结束。
```

#### 当循环

**语法**：
```段言
当 条件：
  语句。
结束。
```

**示例**：
```段言
设 计数 为 零。
当 计数 小于 十：
  打印 计数。
  设 计数 为 计数 加 一。
结束。
```

### 5. 模块系统

**导入模块**：
```段言
导入《模块名》。
从《模块名》导入 符号一 符号二。
```

**导出符号**：
```段言
导出 符号一 符号二。
```

**示例**：
```段言
# 标准库模块
导入《数学工具》。
从《数学工具》导入 平方 绝对值。

# 使用导入的函数
设 结果 为 平方(5)。
打印 结果。

# 导出
段落 加倍 接收 数值：
  返回 数值 乘 二。
结束。

导出 加倍。
```

---

### 6. 异常处理

**语法**：
```段言
尝试：
  语句。
捕获 异常变量：
  处理语句。
最终：
  清理语句。
结束。
```

**示例**：
```段言
尝试：
  设 文件 为 读取文件("data.txt")。
  打印 文件。
捕获 错误：
  打印 "读取失败："。
  打印 错误。
最终：
  打印 "处理完成。"。
结束。
```

**抛出异常**：
```段言
抛出 "错误信息"。
```

## 标点符号使用规则

### 必须使用

1. **中文句号（。）**：语句结束符
2. **冒号（：）**：代码块开始（段落、类、if等）
3. **书名号（《》）**：模块名引用

### 可选使用

1. **中文逗号（，）**：分隔多个参数
2. **英文括号（）**：函数调用（可选）

### 尽量避免

1. **英文分号**：不使用
2. **英文句号**：不使用
3. **花括号**：不使用

## 注意事项

1. **冒号和句号的区别**：
   - 冒号（：）：用于代码块开始（段落、类、if等）
   - 句号（。）：用于语句结束

2. **参数列表**：
   - 参数后使用冒号，不是句号
   - 正确：`段落 平方 接收 数值：`
   - 错误：`段落 平方 接收 数值。`

3. **函数调用**：
   - 推荐使用空格：`平方 甲`
   - 也支持括号：`平方(甲)`

4. **self引用**：
   - 使用"己"关键字
   - 示例：`己.名称` 或 `己.方法`
