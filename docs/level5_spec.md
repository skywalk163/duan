## 二、异常处理

### 2.1 语法

```
尝试：
    可能出错的代码()
捕获 值错误 as e：
    输出("值错误：" + 字符串(e))
捕获 类型错误：
    输出("类型错误")
捕获：
    输出("其他错误")
最终：
    清理资源()
```

### 2.2 抛出语句

```
抛出 "错误消息"
抛出 错误对象
```

### 2.3 异常类型映射

| 中文别名 | Python 类型 |
|---------|-----------|
| 异常 | Exception |
| 值错误 | ValueError |
| 类型错误 | TypeError |
| 键错误 | KeyError |
| 索引错误 | IndexError |
| 除零错误 | ZeroDivisionError |
| 属性错误 | AttributeError |
| 名称错误 | NameError |
| 文件错误 | FileNotFoundError |
| 运行错误 | RuntimeError |

## 三、模块系统

### 3.1 导入语法

```
导入 模块名

```

### 3.2 导出语法

```
# 独立导出语句
导出 函数1
导出 函数2

# 内联导出
段 函数1 导出 接收 参数：
    ...
```

2. 当前目录/模块/
3. 当前目录/modules/
4. 当前目录/test_modules/
5. 标准库目录

## 五、使用示例

### 5.1 异常处理

```
尝试：
    设 结果 为 1 除 0
捕获 除零错误：
    输出("除零错误")
捕获：
    输出("其他错误")
最终：
    输出("清理")
```

### 5.2 模块系统

math_utils.duan:
```
段 加法 导出 接收 a, b：
    返回 a 加 b
结束。
```

main.duan:
```
导入 math_utils
设 r 为 加法(3, 4)
输出(r)
```
