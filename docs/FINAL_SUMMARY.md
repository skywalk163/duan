### 示例代码

**段言代码：**
```段言
类 学生：
  属性 姓名: str
  属性 年龄: int
  
  构造(姓名, 年龄)：
    定义己姓名等于姓名。
    定义己年龄等于年龄。
  
  介绍：
    打印"我叫"加己姓名。
类结束
```

**生成的Python代码：**
```python
class 学生:
    姓名: str
    年龄: int
    
    def __init__(self, 姓名, 年龄):
        self.姓名 = 姓名
        self.年龄 = 年龄
    
    def 介绍(self):
        print(f"我叫{self.姓名}")
```
