# Duan Lite v1.0 语法规范

**版本**: v1.0  
**日期**: 2026-06-30  
**状态**: 稳定（二次自举验证通过）  
**对应实现**: `bootstrap/bootstrap_level3.duan`

---

## 一、设计原则

Duan Lite 是段言的最小可信核心子集，设计目标：

1. **最小性**：尽可能少的语法特性，保持编译器小巧
2. **表达力**：足够编写下一层编译器（Level 4）
3. **稳定性**：二次自举完全一致，可重复验证
4. **可审计**：代码量小，可人工逐行审查

## 二、词法结构

### 2.1 字符集

- 支持 UTF-8 编码
- 标识符支持：中文、英文字母（a-z, A-Z）、数字（0-9）
- 标识符不能以数字开头

### 2.2 关键字（共 29 个）

| 类别 | 关键字 |
|------|--------|
| 变量 | `设`、`为` |
| 函数 | `段`、`段落`、`接收`、`返回` |
| 控制流 | `如果`、`否则`、`当`、`结束` |
| 循环 | `遍历`、`在` |
| 算术 | `加`、`减`、`乘`、`除`、`取模` |
| 比较 | `等于`、`不等于`、`小于`、`大于`、`小于等于`、`大于等于` |
| 布尔 | `且`、`或`、`非` |
| 面向对象 | `类`、`属性`、`己` |

### 2.3 字面量

#### 字符串
- 使用双引号 `"` 包裹：`"hello"`
- 转义表示：
  - `'"'` （单引号+双引号+单引号）表示双引号字符
  - `"'"` （双引号+单引号+双引号）表示单引号字符
- 不支持其他转义序列（`\n`、`\t` 等需直接写入字符）

#### 数字
- 整数：`123`、`456`
- 不支持浮点数（Level 3 限制）

#### 布尔值
- `真` → True
- `假` → False

### 2.4 标点符号

| 符号 | 名称 | 用途 |
|------|------|------|
| `。` | 句号 | 语句结束 |
| `：` | 冒号 | 块开始 |
| `，` | 逗号 | 分隔符（已废弃，用英文逗号） |
| `,` | 英文逗号 | 参数分隔 |
| `(` | 左括号 | 函数调用 |
| `)` | 右括号 | 函数调用 |

### 2.5 注释

- 单行注释：以 `#` 开头，到行尾结束
- 不支持多行注释

### 2.6 缩进

- 使用空格缩进，2 空格为一级
- 块结构（函数、if、while、for、类）通过缩进标识
- 块以 `结束。` 关键字结束

## 三、语法规范

### 3.1 顶层结构

```段言
段落 函数名 接收 参数1, 参数2：
  函数体
结束。

段 函数名 接收 参数1：
  函数体
结束。
```

- `段落` 和 `段` 等价，都是函数定义关键字
- 函数参数用英文逗号 `,` 分隔
- 函数体缩进一级
- 以 `结束。` 结束函数定义

### 3.2 变量声明

```段言
设 变量名 为 表达式。
```

- 使用 `设` 关键字声明变量
- 使用 `为` 关键字赋值
- 变量无需声明类型，动态类型

### 3.3 条件语句

```段言
如果 条件表达式：
  条件为真时执行
结束。

如果 条件表达式：
  条件为真时执行
否则：
  条件为假时执行
结束。
```

- 不支持 `否则若` / `else if`（需用嵌套 if 实现）

### 3.4 循环

#### while 循环
```段言
当 条件表达式：
  循环体
结束。
```

#### for 循环（遍历）
```段言
遍历 变量 在 列表表达式：
  循环体
结束。
```

### 3.5 类定义

```段言
类 类名：
  属性 属性名1。
  属性 属性名2。
  
  段落 方法名 接收 己, 参数1：
    方法体
  结束。
结束。
```

- 使用 `类` 关键字定义类
- 使用 `属性` 关键字声明属性
- 方法第一个参数必须是 `己`（等价于 Python 的 self）
- 不支持继承

### 3.6 返回语句

```段言
返回 表达式。
```

- 用于函数返回值
- 函数可以没有返回语句

### 3.7 函数调用

```段言
函数名(参数1, 参数2)
```

- 参数用英文逗号 `,` 分隔
- 支持嵌套调用

### 3.8 表达式

#### 运算符优先级（从高到低）

| 优先级 | 运算符 | 说明 |
|--------|--------|------|
| 1 | `非` | 一元布尔非 |
| 2 | `乘`、`除`、`取模` | 乘除模 |
| 3 | `加`、`减` | 加减 |
| 4 | `等于`、`不等于`、`小于`、`大于`、`小于等于`、`大于等于` | 比较 |
| 5 | `且` | 逻辑与 |
| 6 | `或` | 逻辑或 |

#### 一元表达式

```段言
非 条件
```

#### 连续运算

同优先级运算符支持连续运算，左结合：

```段言
a 加 b 加 c      # (a + b) + c
x 且 y 且 z      # (x and y) and z
```

## 四、内置函数

Level 3 编译器依赖以下内置函数（由运行时提供）：

### 4.1 列表操作

| 函数 | 说明 |
|------|------|
| `列表创建(...)` | 创建列表 |
| `列表追加(lst, item)` | 向列表追加元素 |
| `列表获取(lst, i)` | 获取列表第 i 个元素 |
| `列表长度(lst)` | 获取列表长度 |

### 4.2 字符串操作

| 函数 | 说明 |
|------|------|
| `字符串长度(s)` | 获取字符串长度 |
| `字符串获取(s, i)` | 获取字符串第 i 个字符 |
| `截取(s, start, end)` | 截取子串 |

### 4.3 其他

| 函数 | 说明 |
|------|------|
| `建(type, value)` | 构建 token |
| `打印(...)` | 输出打印 |

## 六、已知限制

1. 不支持浮点数
2. 不支持字典（可用列表模拟）
3. 不支持异常处理
4. 不支持模块系统
5. 不支持继承
6. 不支持 `否则若` / `else if`
7. 不支持 `break` / `continue`
8. 不支持多行注释
9. 不支持列表/字典字面量（需用函数创建）
10. 不支持字符串插值
11. 错误处理简单，依赖运行时异常
12. 仅支持 Python 代码生成后端

## 七、验证标准

Duan Lite v1.0 必须通过以下验证：

1. **自举测试**：编译器能编译自身源码
2. **二次自举稳定**：第二次与第三次自举输出完全一致
3. **回归测试**：50+ 测试用例全部通过
4. **性能基线**：编译自身 < 0.5 秒
5. **代码审计**：核心函数（词法、语法、代码生成）通过人工审查
