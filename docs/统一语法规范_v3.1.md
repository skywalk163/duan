## 重要变更（v2.0 → v3.1）

| 变更项 | v2.0（旧） | v3.1（新） | 说明 |
|--------|-----------|-----------|------|
| 块结束标记 | `结束。`（必需） | 移除 | 采用纯缩进式语法，类似 Python |
| 句号 | 语句必须以句号结尾 | 句号可选 | 缩进和关键字自然分隔语句 |
| 解构赋值 | 仅支持元组 `(甲, 乙)` | 支持元组和列表 `[首, 余]` | 增加列表解构语法 |
| 范围表达式 | 无 | `1至10` / `1到10步2` | 生成 `range(start, end+1, step)` |
| 无括号调用 | 部分支持 | 完全支持 | 动词调用可省略括号 |

## 核心设计原则

### 1. 纯缩进式语法
使用冒号 `：` 开始块，使用缩进来表示块层级，不再需要「结束」关键字。

```段言
段落 平方 接收 数值：
  返回 数值 乘 数值

如果 甲 大于 10：
  打印 "大于10"
否则：
  打印 "不大于10"

类 动物：
  属性 名称
  段 说话：
    打印 "叫声"
```

### 2. 句号可选
语句结尾的句号 `。` 是可选的，用于增强可读性而非语法必需。

```段言
# 两种写法都正确
设 甲 为 10
设 乙 为 20。
```

### 3. 中文无空格
中文关键字和标识符之间不需要空格，保持中文书写的自然流畅。

```段言
如果甲大于十那么打印甲
```




#### 列表解构



### 2.6 段落定义

#### 标准形式

## 十、语法风格统一表

### 推荐写法 vs 兼容写法

| 不推荐（旧语法） | 推荐（新语法） | 说明 |
|-----------------|---------------|------|
| `结束。` | （省略，用缩进） | 纯缩进式语法 |
| `打印 甲。` | `打印(甲)。` 或 `打印 甲` | 动词调用两种均可 |
| `新建 类名 参数` | `新建 类名(参数)` | 类实例化推荐括号形式 |
| `遍历 甲 于 列表` | `遍历 甲 之 列表` | `之` 是推荐写法 |
| `段 平方(数值)：` | `段落 平方 接收 数值：` | 标准段落定义形式 |
| `使用 表达式 作为 变量` | `使用 表达式 为 变量` | `为` 是标准关键字 |

## 十一、完整示例

### 阶乘函数

```段言
段落 阶乘 接收 n：
  如果 n 小于等于 1：
    返回 1
  返回 n 乘 阶乘(n 减 1)
```

```段言
段落 斐波那契 接收 n：

遍历 i 之 1至10：
  打印 斐波那契(i)
```
