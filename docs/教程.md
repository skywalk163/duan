## 第一章：入门

### 1.1 什么是段言？

段言是一种面向中文用户的编程语言，采用中文关键字和自然语言风格的语法设计。它的目标是让中文用户能够更自然地学习和使用编程。

### 1.2 安装

```bash
# 克隆项目
git clone https://github.com/dumate/duan.git
cd duan

# 安装依赖
pip install antlr4-python3-runtime

# 运行测试验证环境
python test_comprehensive_v2.py
```

### 1.3 第一个程序

创建文件 `hello.duan`：

```段言
打印 "你好，世界！"。
```

运行：

```bash
python cli/duanc.py hello.duan --run
```

输出：
```
你好，世界！
```

### 1.4 使用 REPL

```bash
python -m src.repl.core
```

在 REPL 中可以直接输入代码：

```
段言> 打印 "你好"。
你好
段言> 设 甲 为 10。
段言> 打印 甲。
10
```

## 第二章：变量和数据类型

### 2.1 定义变量

段言支持两种变量声明语法：

```段言
# v1.6 语法（推荐）：设...为...
设 姓名 为 "张三"。
设 年龄 为 25。
设 身高 为 1.75。
设 已婚 为 真。

# v1.5 语法（兼容）：定义...等于...
定义 城市 等于 "北京"。
定义 温度 等于 28。
```

### 2.2 数据类型

| 类型 | 示例 | 说明 |
|------|------|------|
| 整数 | `42` | 整数值 |
| 浮点数 | `3.14` | 小数值 |
| 字符串 | `"你好"` | 文本 |
| 布尔值 | `真`、`假` | 逻辑值 |
| 空值 | `空` | 空值 |

### 2.3 列表

```段言
设 数字列表 为 [1, 2, 3, 4, 5]。
设 字符串列表 为 ["苹果", "香蕉", "橙子"]。

# 访问元素
设 第一个 为 数字列表[0]。  # 1
设 长度 为 长(数字列表)。   # 5

# 追加元素
列表追加 数字列表, 6。

# 遍历列表
遍历 项 之 数字列表：
  打印 项。
结束。
```

### 2.4 字典

```段言
设 学生 为 {"姓名": "张三", "年龄": 25}。

# 访问值
设 名字 为 学生["姓名"]。  # "张三"

# 设置值
字典设置 学生, "成绩", 95。
```

## 第三章：运算符

### 3.1 算术运算

```段言
设 a 为 10 加 5。     # 15
设 b 为 10 减 5。     # 5
设 c 为 10 乘 5。     # 50
设 d 为 10 除 5。     # 2.0
```

也支持复合运算符：

```段言
甲 加上 乙    # 同"加"
甲 减去 乙    # 同"减"
甲 乘以 乙    # 同"乘"
甲 除以 乙    # 同"除"
```

### 3.2 比较运算

```段言
设 x 为 10。
设 结果1 为 x 等于 10。       # 真
设 结果2 为 x 大于 5。        # 真
设 结果3 为 x 小于 20。       # 真
设 结果4 为 x 大于等于 10。   # 真
设 结果5 为 x 小于等于 10。   # 真
设 结果6 为 x 不等于 5。      # 真
```

### 3.3 逻辑运算

```段言
设 a 为 真。
设 b 为 假。

设 结果1 为 a 且 b。   # 假
设 结果2 为 a 或 b。   # 真
设 结果3 为 非 a。     # 假
```

## 第四章：控制流程

### 4.1 条件语句

```段言
设 年龄 为 18。

如果 年龄 大于 18：
  打印 "成年人"。
否则若 年龄 等于 18：
  打印 "刚成年"。
否则：
  打印 "未成年"。
结束。
```

### 4.2 当循环

```段言
设 计数 为 0。

当 计数 小于 5：
  打印 计数。
  设 计数 为 计数 加 1。
结束。
```

### 4.3 遍历循环

```段言
设 水果 为 ["苹果", "香蕉", "橙子"]。

遍历 水果项 之 水果：
  打印 水果项。
结束。
```

### 4.4 跳出和跳过

```段言
设 计数 为 0。

当 计数 小于 10：
  如果 计数 等于 5：
    跳出。  # 跳出循环
  结束。
  
  如果 计数 等于 3：
    跳过。  # 跳过当前迭代
  结束。
  
  打印 计数。
  设 计数 为 计数 加 1。
结束。
```

### 4.5 异常处理

```段言
# 尝试-捕获
尝试：
  设 结果 为 10 除 0。
  打印 结果。
捕获 错误：
  打印 "出错了："。
  打印 错误。
结束。

# 带最终块的异常处理
尝试：
  设 文件 为 打开文件 "data.txt"。
  打印 文件。
捕获 错误：
  打印 "文件读取失败"。
最终：
  打印 "清理完成"。
结束。

# 抛出异常
段落 检查年龄 接收 年龄：
  如果 年龄 小于 0：
    抛出 "年龄不能为负数"。
  结束。
结束。
```

## 第五章：段落（函数）

### 5.1 定义段落

段落是段言中的函数概念。使用 `段落...接收...` 语法：

```段言
段落 平方 接收 数值：
  返回 数值 乘 数值。
结束。

段落 求平均值 接收 甲, 乙, 丙：
  设 总和 为 甲 加 乙 加 丙。
  返回 总和 除 3。
结束。

段落 问候：
  打印 "你好，世界！"。
结束。
```

### 5.2 调用段落

```段言
# 直接调用
打印 "Hello"。
问候()。

# 接收返回值
设 结果 为 平方(5)。       # 25
设 平均 为 求平均值(10, 20, 30)。  # 20.0
```

### 5.3 递归段落

```段言
段落 阶乘 接收 数值：
  如果 数值 小于等于 1：
    返回 1。
  结束。
  返回 数值 乘 阶乘(数值 减 1)。
结束。

打印 阶乘(5)。  # 120
```

### 5.4 斐波那契数列

```段言
段落 斐波那契 接收 数：
  如果 数 小于等于 2：
    返回 1。
  结束。
  返回 斐波那契(数 减 1) 加 斐波那契(数 减 2)。
结束。

设 计数 为 1。
当 计数 小于等于 10：
  打印 斐波那契(计数)。
  设 计数 为 计数 加 1。
结束。
```

## 第六章：类和对象

### 6.1 定义类

```段言
类 动物：
  段落 叫声：
    打印 "动物叫声"。
  结束。
结束。
```

### 6.2 属性和构造函数

```段言
类 学生：
  属性 姓名。
  属性 年龄。
  属性 成绩。
  
  构造 接收 姓名, 年龄：
    己姓名 为 姓名。
    己年龄 为 年龄。
  结束。
结束。
```

### 6.3 方法定义

```段言
类 学生：
  属性 姓名。
  属性 年龄。
  属性 成绩。
  
  构造 接收 姓名, 年龄：
    己姓名 为 姓名。
    己年龄 为 年龄。
    己成绩 为 []。
  结束。
  
  段落 添加成绩 接收 分数：
    列表追加 己成绩, 分数。
  结束。
  
  段落 平均成绩：
    如果 长(己成绩) 等于 0：
      返回 0。
    结束。
    
    设 总分 为 0。
    遍历 分 之 己成绩：
      设 总分 为 总分 加 分。
    结束。
    
    返回 总分 除 长(己成绩)。
  结束。
  
  段落 介绍：
    打印 "姓名："。
    打印 己姓名。
    打印 "平均成绩："。
    打印 平均成绩()。
  结束。
结束。
```

### 6.4 创建对象和使用

```段言
# 创建对象
设 张同学 为 新建 学生 "张三", 18。

# 调用方法
张同学.添加成绩(85)。
张同学.添加成绩(90)。
张同学.介绍()。
```

### 6.5 继承

```段言
类 动物：
  段落 叫声：
    打印 "动物叫声"。
  结束。
结束。

类 狗 继承 动物：
  属性 名称。
  
  构造 接收 名称：
    己名称 为 名称。
  结束。
  
  段落 叫声：
    打印 "汪汪汪"。
  结束。
  
  段落 介绍：
    打印 "我是小狗"。
    打印 己名称。
  结束。
结束。

类 猫 继承 动物：
  段落 叫声：
    打印 "喵喵喵"。
  结束。
结束。

设 小狗 为 新建 狗 "旺财"。
小狗.叫声()。    # 汪汪汪
小狗.介绍()。    # 我是小狗 / 旺财

设 小猫 为 新建 猫。
小猫.叫声()。    # 喵喵喵
```

`己` 关键字（self 引用）：

| 用法 | 说明 |
|------|------|
| `己名称` | 访问自身属性 |
| `己名称 为 新值。` | 设置自身属性 |
| `己方法()` | 调用自身方法 |

## 第七章：模块系统

### 7.1 创建模块

创建文件 `数学工具.duan`：

```段言
段落 平方 接收 数值：
  返回 数值 乘 数值。
结束。

段落 阶乘 接收 数值：
  如果 数值 小于等于 1：
    返回 1。
  结束。
  返回 数值 乘 阶乘(数值 减 1)。
结束。

# 导出函数
导出 平方 阶乘。
```

### 7.2 导入模块

```段言
# 方式一：导入整个模块
导入 数学工具。

# 方式二：从模块导入特定函数
从 数学工具 导入 平方 阶乘。

# 方式三：带别名
导入 数学工具 为 数学。

# 使用
打印 平方(5)。      # 25
打印 阶乘(5)。      # 120
```

### 7.3 模块搜索路径

段言按以下顺序搜索模块：

1. 当前文件所在目录
2. 命令行指定的搜索路径
3. `DUAN_PATH` 环境变量指定的路径

## 第八章：实战项目

### 8.1 计算器

```段言
类 计算器：
  属性 结果。
  
  构造 接收 初始值：
    己结果 为 初始值。
  结束。
  
  段落 加 接收 数值：
    己结果 为 己结果 加 数值。
  结束。
  
  段落 减 接收 数值：
    己结果 为 己结果 减 数值。
  结束。
  
  段落 乘 接收 数值：
    己结果 为 己结果 乘 数值。
  结束。
  
  段落 除 接收 数值：
    如果 数值 不等于 0：
      己结果 为 己结果 除 数值。
    否则：
      打印 "错误：除数不能为零"。
    结束。
  结束。
  
  段落 清零：
    己结果 为 0。
  结束。
  
  段落 获取结果：
    返回 己结果。
  结束。
结束。

设 计算器1 为 新建 计算器 0。
计算器1.加(10)。
计算器1.乘(2)。
计算器1.减(5)。
打印 计算器1.获取结果()。  # 15
```

### 8.3 简单游戏

```段言
类 角色：
  属性 名称。
  属性 生命值。
  属性 攻击力。
  
  构造 接收 名称：
    己名称 为 名称。
    己生命值 为 100。
    己攻击力 为 10。
  结束。
  
  段落 攻击 接收 目标：
    目标.受伤(己攻击力)。
    打印 己名称。
    打印 "攻击了"。
    打印 目标.名称。
  结束。
  
  段落 受伤 接收 伤害：
    己生命值 为 己生命值 减 伤害。
    如果 己生命值 小于 0：
      己生命值 为 0。
    结束。
  结束。
  
  段落 是否存活：
    返回 己生命值 大于 0。
  结束。
结束。

设 玩家 为 新建 角色 "英雄"。
设 怪物 为 新建 角色 "怪物"。

当 玩家.是否存活() 且 怪物.是否存活()：
  玩家.攻击(怪物)。
  如果 怪物.是否存活()：
    怪物.攻击(玩家)。
  结束。
结束。

如果 玩家.是否存活()：
  打印 "玩家胜利！"。
否则：
  打印 "怪物胜利！"。
结束。
```
