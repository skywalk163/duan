## 第一章：入门

### 1.1 什么是段言？

段言是一种面向中文用户的编程语言，采用中文关键字和自然语言风格的语法设计。它的目标是让中文用户能够更自然地学习和使用编程。

### 1.2 安装

```bash
# 克隆项目
git clone https://github.com/your-repo/duan.git

# 安装依赖
pip install antlr4-python3-runtime
```

### 1.3 第一个程序

创建文件 `hello.duan`：

```段言
打印("你好，世界！")。
```

运行：

```bash
python duan_cli.py run hello.duan
```

输出：
```
你好，世界！
```

## 第二章：变量和数据类型

### 2.1 定义变量

使用 `定义` 关键字声明变量：

```段言
定义 姓名 等于 "张三"。
定义 年龄 等于 25。
定义 身高 等于 1.75。
定义 已婚 等于 真。
```

### 2.2 数据类型

段言支持以下基本数据类型：

| 类型 | 示例 |
|------|------|
| 整数 | `42` |
| 浮点数 | `3.14` |
| 字符串 | `"你好"` |
| 布尔值 | `真`, `假` |
| 空值 | `空` |

### 2.3 列表

列表是有序的元素集合：

```段言
定义 数字列表 等于 [1, 2, 3, 4, 5]。
定义 字符串列表 等于 ["苹果", "香蕉", "橙子"]。

# 访问元素
定义 第一个 等于 数字列表[0]。  # 1
定义 长度 等于 listLen(数字列表)。  # 5
```

### 2.4 字典

字典是键值对的集合：

```段言
定义 学生 等于 _典("姓名": "张三", "年龄": 25)。

# 访问值
定义 名字 等于 学生["姓名"]。  # "张三"
```

## 第三章：运算符

### 3.1 算术运算

段言使用中文运算符：

```段言
定义 a 等于 10 加 5。    # 15
定义 b 等于 10 减 5。    # 5
定义 c 等于 10 乘 5。    # 50
定义 d 等于 10 除 5。    # 2.0
定义 e 等于 10 模 3。    # 1
定义 f 等于 2 幂 10。    # 1024
```

### 3.2 比较运算

```段言
定义 x 等于 10。

定义 结果1 等于 x 等于 10。      # 真
定义 结果2 等于 x 大于 5。       # 真
定义 结果3 等于 x 小于 20。      # 真
定义 结果4 等于 x 大于等于 10。  # 真
定义 结果5 等于 x 小于等于 10。  # 真
定义 结果6 等于 x 不等于 5。     # 真
```

### 3.3 逻辑运算

```段言
定义 a 等于 真。
定义 b 等于 假。

定义 结果1 等于 a 且 b。   # 假
定义 结果2 等于 a 或 b。   # 真
定义 结果3 等于 非 a。     # 假
```

## 第四章：控制流程

### 4.1 条件语句

使用 `如果` 进行条件判断：

```段言
定义 年龄 等于 18。

如果 年龄 大于 18 那么:
    打印("成年人")。
否则如果 年龄 等于 18 那么:
    打印("刚成年")。
否则:
    打印("未成年")。
结束。
```

### 4.2 循环语句

使用 `当` 进行循环：

```段言
定义 i 等于 0。

当 i 小于 10:
    打印(i)。
    i 等于 i 加 1。
结束。
```

### 4.3 遍历循环

使用 `遍历` 遍历列表：

```段言
定义 列表 等于 [1, 2, 3, 4, 5]。

遍历 元素 于 列表:
    打印(元素)。
结束。
```

### 4.4 跳出和跳过

```段言
定义 i 等于 0。

当 i 小于 10:
    如果 i 等于 5 那么:
        跳出。  # 跳出循环
    结束。
    
    如果 i 等于 3 那么:
        跳过。  # 跳过当前迭代
    结束。
    
    打印(i)。
    i 等于 i 加 1。
结束。
```

## 第五章：段落（函数）

### 5.1 定义段落

段落是段言中的函数概念：

```段言
《加》段(a, b):
    返回 a 加 b。
结束。
```

### 5.2 调用段落

```段言
定义 结果 等于 《加》(3, 5)。
打印(结果)。  # 8
```

### 5.3 递归段落

```段言
《阶乘》段(n):
    如果 n 小于等于 1 那么:
        返回 1。
    结束。
    返回 n 乘 《阶乘》(n 减 1)。
结束。

定义 结果 等于 《阶乘》(5)。
打印(结果)。  # 120
```

### 5.4 多参数段落

```段言
《求平均值》段(a, b, c):
    定义 总和 等于 a 加 b 加 c。
    返回 总和 除 3。
结束。

定义 平均 等于 《求平均值》(10, 20, 30)。
打印(平均)。  # 20.0
```

## 第六章：类和对象

### 6.1 定义类

```段言
《人》类:
    定义 姓名 等于 ""。
    定义 年龄 等于 0。
    
    《初始化》方法(姓名参数, 年龄参数):
        姓名 等于 姓名参数。
        年龄 等于 年龄参数。
    结束。
    
    《说话》方法():
        打印(姓名 + "说你好")。
    结束。
    
    《成长》方法(年数):
        年龄 等于 年龄 加 年数。
    结束。
结束。
```

### 6.2 创建对象

```段言
定义 张三 等于 新 人("张三", 25)。
张三之说话()。  # 张三说你好
```

### 6.3 访问属性和方法

```段言
# 访问属性
定义 名字 等于 张三之姓名。

# 调用方法
张三之成长(3)。
打印(张三之年龄)。  # 28
```

### 6.4 继承

```段言
《动物》类:
    定义 名称 等于 ""。
    
    《叫声》方法():
        打印("叫声")。
    结束。
结束。

《狗》类 继承 动物:
    《叫声》方法():
        打印("汪汪")。
    结束。
结束。

定义 小狗 等于 新 狗()。
小狗之叫声()。  # 汪汪
```

## 第七章：模块系统

### 7.1 创建模块

创建文件 `modules/my_module.duan`：

```段言
导出 《我的段落》。
导出 我的类。

《我的段落》段(x):
    返回 x 乘 2。
结束。

《我的类》类:
    定义 值 等于 0。
    
    《初始化》方法(v):
        值 等于 v。
    结束。
结束。
```

### 7.2 导入模块

```段言
从 my_module 导入 《我的段落》, 我的类。

定义 结果 等于 《我的段落》(5)。
打印(结果)。  # 10

定义 对象 等于 新 我的类(42)。
打印(对象之值)。  # 42
```

### 7.3 模块搜索路径

段言按以下顺序搜索模块：

1. 当前文件所在目录
2. 命令行指定的搜索路径
3. `DUAN_PATH` 环境变量指定的路径

## 第八章：实战项目

### 8.1 计算器

创建文件 `calculator.duan`：

```段言
《计算器》类:
    定义 结果 等于 0。
    
    《加》方法(x):
        结果 等于 结果 加 x。
    结束。
    
    《减》方法(x):
        结果 等于 结果 减 x。
    结束。
    
    《乘》方法(x):
        结果 等于 结果 乘 x。
    结束。
    
    《除》方法(x):
        如果 x 不等于 0 那么:
            结果 等于 结果 除 x。
        否则:
            打印("错误: 除数不能为零")。
        结束。
    结束。
    
    《清零》方法():
        结果 等于 0。
    结束。
    
    《获取结果》方法():
        返回 结果。
    结束。
结束。

定义 计算器1 等于 新 计算器()。
计算器1之加(10)。
计算器1之乘(2)。
计算器1之减(5)。
打印(计算器1之获取结果())。  # 15
```

### 8.2 学生管理系统

```段言
《学生》类:
    定义 姓名 等于 ""。
    定义 年龄 等于 0。
    定义 成绩 等于 []。
    
    《初始化》方法(姓名参数, 年龄参数):
        姓名 等于 姓名参数。
        年龄 等于 年龄参数。
    结束。
    
    《添加成绩》方法(分数):
        listAppend(成绩, 分数)。
    结束。
    
    《平均成绩》方法():
        如果 listLen(成绩) 等于 0 那么:
            返回 0。
        结束。
        
        定义 总分 等于 0。
        遍历 分 于 成绩:
            总分 等于 总分 加 分。
        结束。
        
        返回 总分 除 listLen(成绩)。
    结束。
    
    《打印信息》方法():
        打印("姓名: " + 姓名)。
        打印("年龄: " + _串化(年龄))。
        打印("平均成绩: " + _串化(平均成绩()))。
    结束。
结束。

定义 学生1 等于 新 学生("张三", 18)。
学生1之添加成绩(85)。
学生1之添加成绩(90)。
学生1之添加成绩(78)。
学生1之打印信息()。
```

### 8.3 简单游戏

```段言
《角色》类:
    定义 名称 等于 ""。
    定义 生命值 等于 100。
    定义 攻击力 等于 10。
    
    《初始化》方法(名称参数):
        名称 等于 名称参数。
    结束。
    
    《攻击》方法(目标):
        目标之受伤(攻击力)。
        打印(名称 + "攻击了" + 目标之名称 + "，造成" + _串化(攻击力) + "点伤害")。
    结束。
    
    《受伤》方法(伤害):
        生命值 等于 生命值 减 伤害。
        如果 生命值 小于 0 那么:
            生命值 等于 0。
        结束。
    结束。
    
    《是否存活》方法():
        返回 生命值 大于 0。
    结束。
结束。

定义 玩家 等于 新 角色("英雄")。
定义 怪物 等于 新 角色("怪物")。

当 玩家之是否存活() 且 怪物之是否存活():
    玩家之攻击(怪物)。
    如果 怪物之是否存活() 那么:
        怪物之攻击(玩家)。
    结束。
结束。

如果 玩家之是否存活() 那么:
    打印("玩家胜利！")。
否则:
    打印("怪物胜利！")。
结束。
```
