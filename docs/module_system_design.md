# 段言模块系统设计文档

**版本**: v0.9.0  
**日期**: 2026-06-10  
**状态**: 设计中

---

## 一、设计目标

1. **代码组织**：支持将编译器拆分为多个模块文件
2. **命名空间**：避免不同模块间的命名冲突
3. **依赖管理**：明确模块间的依赖关系
4. **自举支持**：为段言编译器自举提供基础设施

---

## 二、语法设计

### 2.1 基本导入

```段言
# 导入整个模块
导入《模块名》。

# 导入并使用
导入《数学库》。
定义结果等于《数学库.平方根》参数16。
```

### 2.2 选择性导入

```段言
# 从模块导入特定函数
从《数学库》导入《平方根》，《绝对值》。

# 直接使用
定义结果等于《平方根》参数16。
```

### 2.3 别名导入

```段言
# 模块别名
导入《数学库》为数学。

# 使用别名
定义结果等于《数学.平方根》参数16。

# 函数别名
从《数学库》导入《平方根》为开方。
```

### 2.4 导出

```段言
# 导出函数
导出《函数名》。

# 导出多个
导出《函数1》，《函数2》，《函数3》。

# 导出所有段落
导出全部。
```

### 2.5 完整示例

**文件：math_utils.duan**
```段言
# 数学工具模块

导出《平方》，《立方》。

《平方》段(数)：
  返回数乘数。

《立方》段(数)：
  返回数乘数乘数。

# 内部函数，不导出
《辅助计算》段(数)：
  返回数加一。
```

**文件：main.duan**
```段言
# 主程序

从《math_utils》导入《平方》，《立方》。

定义甲等于5。
定义平方结果等于《平方》参数甲。
定义立方结果等于《立方》参数甲。

打印平方结果。
打印立方结果。
```

## 三、语义规则

### 3.1 模块查找规则

1. **当前目录优先**：先在当前目录查找 `<模块名>.duan`
2. **标准库路径**：在 `DUAN_PATH` 环境变量指定的路径查找
3. **错误处理**：找不到模块时报错

### 3.2 命名空间规则

```段言
# 方式1：模块名作为前缀
导入《数学》。
定义结果等于《数学.平方根》参数16。

# 方式2：直接使用（选择性导入）
从《数学》导入《平方根》。
定义结果等于《平方根》参数16。
```

### 3.3 导出规则

- 默认情况下，段落不导出（模块私有）
- 使用 `导出` 语句显式导出
- `导出全部` 导出所有段落

### 3.4 循环依赖

- 检测循环依赖并报错
- 错误信息：`错误：检测到循环依赖：A → B → C → A`

## 四、实现架构

### 4.1 新增组件

```
src/
├── module_resolver.py    # 模块解析器
├── module_loader.py      # 模块加载器
├── symbol_table.py       # 符号表（扩展）
└── ...
```

### 4.2 编译流程

```
主模块 (main.duan)
    ↓
模块解析器 (查找导入语句)
    ↓
依赖图构建
    ↓
拓扑排序
    ↓
逐个编译模块
    ↓
链接符号
    ↓
生成Python代码
```

### 4.3 核心类设计

```python
# module_resolver.py

@dataclass
class Module:
    """模块信息"""
    name: str              # 模块名
    path: str              # 文件路径
    imports: List[Import]  # 导入列表
    exports: Set[str]      # 导出符号
    symbols: Dict[str, Any] # 所有符号

@dataclass  
class Import:
    """导入信息"""
    module_name: str       # 模块名
    symbols: List[str]     # 导入的符号（空表示导入全部）
    alias: Optional[str]   # 别名

class ModuleResolver:
    """模块解析器"""
    
    def resolve(self, main_file: str) -> List[Module]:
        """解析主文件及其所有依赖"""
        
    def find_module(self, module_name: str, from_dir: str) -> str:
        """查找模块文件路径"""
        
    def detect_circular_dependency(self, graph: Dict) -> List[str]:
        """检测循环依赖"""
```

## 七、错误处理

### 7.1 错误类型

```python
class ModuleError(DuanError):
    """模块相关错误"""

class ModuleNotFoundError(ModuleError):
    """模块未找到"""

class CircularDependencyError(ModuleError):
    """循环依赖"""

class SymbolNotFoundError(ModuleError):
    """符号未找到"""
```

### 7.2 错误信息示例

```
错误：模块未找到：'数学库'
  位置：main.duan:3
  搜索路径：
    - ./数学库.duan (不存在)
    - /usr/lib/duan/数学库.duan (不存在)

错误：检测到循环依赖
  main.duan → utils.duan → math.duan → main.duan

错误：符号未导出
  模块 '数学库' 未导出符号 '内部函数'
  位置：main.duan:5
```
