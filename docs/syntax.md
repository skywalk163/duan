|------|------|------|

## 泛型

### 泛型函数

使用类型参数实现参数化类型，编写可复用的代码：

```段言
# 单类型参数
段落 恒等[T] 参数 x：
    返回 x

设 结果1 为 恒等(42)      # x 被推断为 数字
设 结果2 为 恒等("你好")  # x 被推断为 字符串

# 多类型参数
段落 映射[T, U] 参数 列表 函数：
    返回 列表
```

### 泛型类

为类定义类型参数：

```段言
# 泛型栈
类 栈[T]：
    属性 数据

    构造：
        己数据 为 []

    段落 入栈 接收 值：
        己数据之追加 值

    段落 出栈：
        返回 己数据之弹出

# 使用泛型类
设 整数栈 为 栈()
设 字符串栈 为 栈()
```

### 泛型方法

在类的方法中使用泛型：

```段言
类 工具：
    段落 转换[T] 接收 值：
        返回 值

    段落 双重[T] 接收 x：
        返回 x 加 x
```

### 泛型约束

类型参数在编译后采用类型擦除策略，运行时与 Python 动态类型完全兼容。
