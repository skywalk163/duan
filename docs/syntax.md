# 语法规范

## 基础语法

### 注释

```段言
# 这是单行注释
```

### 变量声明

```段言
# 显式声明
设 年龄 为 25

# 隐式声明（自动推断类型）
年龄 等于 25
```

### 数据类型

| 类型 | 示例 | 说明 |
|------|------|------|
| 数字 | `42`, `3.14` | 整数和浮点数 |
| 字符串 | `"你好"` | 字符串 |
| 布尔 | `真`, `假` | 布尔值 |
| 列表 | `[1, 2, 3]` | 列表 |
| 字典 | `{"a": 1}` | 字典 |
| 空值 | `空` | 空值 |

### 运算符

```段言
# 算术运算符
结果 等于 10 加 5    # 加
结果 等于 10 减 5    # 减
结果 等于 10 乘 5    # 乘
结果 等于 10 除 5    # 除
结果 等于 10 模 3    # 模
结果 等于 2 幂 3     # 幂

# 比较运算符
真 或 假             # 大于
小 于 大             # 小于
甲 等于 乙           # 等于
甲 不等于 乙         # 不等于
```

## 控制流

### 条件语句

```段言
如果 条件 那么：
    打印 "条件成立"
否则：
    打印 "条件不成立"
```

### 循环

```段言
# while 循环
当 计数 小于 10：
    打印 计数
    计数 等于 计数 加 1

# for 循环（遍历）
遍历 项 在 列表：
    打印 项
```

## 函数定义

```段言
段落 加法 接收 甲 加 乙：
    返回 甲 加 乙

# 带默认参数
段落 问候 接收 名字 等于 "世界"：
    打印 "你好，" 加 名字 加 "！"
```

## 类定义

```段言
类 动物：
    段落 初始化 接收 名称：
        己.名称 等于 名称

    段落 叫：
        打印 己.名称 加 " 在叫"
```

## 异常处理

```段言
尝试：
    可能抛出异常的代码
捕获 错误：
    处理错误
最终：
    无论是否异常都执行
```

## 导入导出

```段言
# 导入模块
导入 数学

# 导出
导出 我的函数
导出 我的类
```

## 字符串插值

```段言
名字 等于 "小明"
打印 f"你好，{名字}！"
```
