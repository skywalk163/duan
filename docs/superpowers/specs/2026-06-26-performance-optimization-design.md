## 1. 目标

### 1.1 编译速度优化
- 词法分析器：字符分类查表 + 正则预编译
- 解析器：AST 节点 `__slots__` + 回溯消除
- 代码生成器：列表 join + 缩进缓存
- 增量编译：基于文件 mtime 的解析缓存

- 死代码消除：删除不可达代码
- 局部变量优化：避免 globals() 查找
- 循环不变量外提：提升循环性能

## 3. 编译速度优化详情

### 3.1 词法分析器优化

**字符分类查表**：
- 用 `bytes.maketrans` 构建 256 字节的字符分类表
- O(1) 判断字符类型（汉字、数字、字母、符号）
- 汉字范围用 codepoint 直接比较（已有优化，保持）

**正则预编译**：
- 所有 `re.compile` 移到模块级
- 字符串数字识别用预编译正则
- 标识符匹配预编译

**关键词匹配**：
- 双字关键词用 `set` 查找（已用 frozenset，保持）
- 单字关键词直接查表

### 3.2 解析器优化

**AST 节点 `__slots__`**：
- 所有 `@dataclass` AST 节点添加 `__slots__ = ()`
- 减少每个实例的内存开销
- 属性访问更快（跳过 `__dict__` 查找）

**回溯消除**：
- 识别当前回溯点（`_try_parse` 模式）
- 改为 lookahead 1~2 token 的预测分析
- 重点优化：表达式解析、函数调用 vs 成员访问

**增量编译缓存**：
- 缓存键：文件路径 + mtime + 文件大小
- 缓存值：解析后的 AST
- 缓存位置：`__pycache__/duan_*.pkl`
- 多模块项目编译时自动启用

### 3.3 代码生成器优化

**字符串构建**：
- 用 `output_lines: List[str]` 收集所有行
- 最后 `'\n'.join(output_lines)` 一次性输出
- 替代当前的逐行 `+=` 拼接

**缩进缓存**：
- 预生成 `_indent_cache: Dict[int, str]`
- `get_indent(level)` 直接查表
- 避免重复生成缩进字符串

**内置映射预计算**：
- `builtin_map` 改为模块级只读字典
- 常用函数名预存，避免运行时查找

## 4. 运行速度优化详情

### 4.1 常量折叠（Constant Folding）

**适用场景**：
- 算术表达式：`1 + 2 * 3` → `7`
- 字符串拼接：`"hello" + "world"` → `"helloworld"`
- 布尔运算：`真 且 假` → `假`
- 列表/字典字面量（全常量）

**实现方式**：
- 后序遍历 AST
- 二元运算节点，若两个操作数都是常量字面量，计算结果替换节点
- 一元运算同理

**边界**：
- 不折叠可能有副作用的表达式（函数调用等）
- 除零错误保留到运行时

### 4.2 死代码消除（Dead Code Elimination）

**消除场景**：
- `如果 假：` 或 `如果 真：否则：` → 只保留可达分支
- `返回` / `抛出` 后的语句删除
- 循环内 `返回` 后的语句删除
- `当 假：` 整个循环删除

**实现方式**：
- 语句列表遍历，遇到终止语句（return/throw/break/continue）截断后续语句
- If 语句条件为常量时，折叠分支

### 4.3 局部变量优化

**优化内容**：
- 函数内所有变量声明保持在局部作用域
- `_duan_builtin` 常用函数在模块开头局部绑定：
  ```python
  _duan_append = _duan_builtin['列表追加']
  ```
- 避免运行时字典查找

### 4.4 循环不变量外提（Loop Invariant Code Motion）

**适用场景**：
- `遍历` 和 `当` 循环
- 循环体内不变的表达式（不依赖循环变量）
- 纯函数调用（已知无副作用）

**实现方式**：
- 分析循环体中每个表达式的依赖
- 不依赖循环变量且无副作用的提到循环前
- 保守策略：只外提纯字面量和简单运算

## 7. 风险与权衡

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 优化器引入新 bug | 高 | 充分测试，优化可选开关 |
| 增量缓存失效 | 中 | 用 mtime + size 双重校验 |
| 常量折叠改变错误时机 | 中 | 除零等运行时错误保留到运行时 |
| `__slots__` 破坏动态属性 | 低 | 审查所有节点，确无动态属性使用 |
