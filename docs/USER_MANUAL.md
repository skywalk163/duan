## 快速开始

### 安装

#### 从 PyPI 安装（推荐）

```bash
pip install duan
```

安装后即可使用 `duan` 命令：
```bash
duan --version
duan --help
```

#### 从源码安装

```bash
git clone https://github.com/skywalk163/duan.git
cd duan
pip install -e .
```

### 第一个程序

创建 `hello.duan` 文件：

```段言
打印 "你好，世界！"
```

运行：

```bash
duan run hello.duan
```

输出：
```
你好，世界！
```

### 编译为 Python 文件

```bash
duan compile hello.duan -o hello.py
```

### 编译为 Windows EXE

```bash
# 需要先安装 PyInstaller
pip install pyinstaller

# 编译为 exe
duan compile hello.duan -o hello.exe
```

**原理：** 先将段言代码编译为 Python 代码，再使用 PyInstaller 打包为独立的 .exe 可执行文件。

**注意：** 仅适用于 Windows 平台，生成的 exe 文件无需 Python 环境即可运行。

## 基础语法

### 注释

段言支持以下注释语法：

```段言
# 这是单行注释（推荐使用）
// 这也是单行注释
注：这也是注释（以"注"开头）
```

段言支持三种注释风格，可以灵活使用。

### 语句结束

每个语句以句号 `。` 结束。

```段言
定义甲等于123。
打印甲。
```

### 标点符号

| 中文符号 | 英文符号 | 用途 |
|---------|---------|-----|
| `。` | `.` | 语句结束 |
| `，` | `,` | 参数分隔、管道操作 |
| `：` | `:` | 块开始、类型注解 |
| `（` `）` | `(` `)` | 括号 |
| `【` `】` | `[` `]` | 列表 |
| `「` `」` | `"` `"` | 字符串 |

## 变量与数据类型

### 变量声明

**语法：** `定义 变量名 等于 值。`

```段言
定义年龄等于25。
定义姓名等于"张三"。
定义成绩等于95.5。
```

### 数据类型

**数字类型：**
```段言
定义整数等于123。
定义浮点等于3.14。
定义中文数字等于三加五。  # 8
```

**字符串：**
```段言
定义消息等于"你好世界"。
```

**布尔值：**
```段言
定义正确等于真。
定义错误等于假。
```

**列表：**
```段言
定义列表等于1，2，3，4，5。
```

### 算术运算

```段言
定义甲等于10。
定义乙等于3。

定义和等于甲加乙。      # 13
定义差等于甲减乙。      # 7
定义积等于甲乘乙。      # 30
定义商等于甲除乙。      # 3.333...
定义余等于甲取余乙。    # 1
```

### 比较运算

```段言
如果甲大于乙那么打印"甲大"。
如果甲小于乙那么打印"甲小"。
如果甲等于乙那么打印"相等"。
```

## 控制流程

### 条件语句

**基础语法：**
```段言
如果条件那么执行语句。
```

**示例：**
```段言
定义分数等于85。

如果分数大于等于90那么打印"优秀"。
否则如果分数大于等于80那么打印"良好"。
否则如果分数大于等于70那么打印"中等"。
否则打印"及格"。
```

### 循环语句

**遍历循环：**
```段言
定义列表等于1，2，3，4，5。

遍历列表中的元素：
  打印元素。
```

**当循环（while）：**
```段言
定义计数等于1。

当计数小于等于5：
  打印计数。
  定义计数等于计数加1。
```

**循环控制：**
```段言
遍历列表中的元素：
  如果元素等于3那么跳出。
  打印元素。
```

## 函数定义

### 基础函数

**语法：** `《函数名》段(参数)：函数体。`

```段言
《加法》段(甲, 乙)：
  返回甲加乙。

定义结果等于《加法》参数三和五。
打印结果。  # 8
```

### 递归函数

**阶乘：**
```段言
《阶乘》段(数)：
  如果数小于等于1那么返回1。
  返回数乘《阶乘》参数数减1。

定义结果等于《阶乘》参数5。
打印结果。  # 120
```

**斐波那契：**
```段言
《斐波那契》段(数)：
  如果数小于等于2那么返回1。
  返回《斐波那契》参数数减1加《斐波那契》参数数减2。

定义结果等于《斐波那契》参数10。
打印结果。  # 55
```

## 类与对象

### 类与对象

### 类定义

**语法：**
```
类 类名：
    属性 属性名。
    段落 方法名：
        # 方法体
```

**示例：**
```段言
类 学生：
    属性 姓名。
    属性 年龄。

    构造 接收 姓名, 年龄：
        己姓名 为 姓名。
        己年龄 为 年龄。

    段落 介绍：
        打印 "我叫" 加 己姓名 加 "，今年" 加 己年龄 加 "岁。".

学生1 等于 新建 学生("张三", 20)。
学生1.介绍()。
```

### 继承

```段言
类 大学生 继承 学生：
    属性 专业。

    构造 接收 姓名, 年龄, 专业：
        父.构造(姓名, 年龄)。
        己专业 为 专业。

    段落 介绍：
        打印 "我叫" 加 己姓名 加 "，专业是" 加 己专业 加 "。"。
```

### 访问修饰符

段言支持三种访问级别：

| 修饰符 | 说明 | 生成代码 |
|--------|------|----------|
| `公有` (默认) | 任何地方可访问 | 无前缀 |
| `私有` | 仅在类内部可访问 | 加 `_` 前缀 |
| `保护` | 类及子类可访问 | 加 `_` 前缀（Python约定）|

```段言
类 账户：
    公有 属性 用户名。
    私有 属性 密码。
    保护 属性 余额。

    构造 接收 用户名, 密码：
        己用户名 为 用户名。
        己密码 为 密码。
        己余额 为 零。

    公有 段落 获取用户名：
        返回 己用户名。

    私有 段落 验证密码 接收 输入密码：
        返回 己密码 等于 输入密码。
```

### 静态属性和方法

使用 `静态` 修饰符定义类级别的属性和方法：

```段言
类 计数器：
    静态 属性 总数 等于 零。

    构造 接收 名称：
        己名称 为 名称。
        计数器.总数 等于 计数器.总数 加 一。

    静态 段落 获取总数：
        返回 计数器.总数。
```

### 属性访问

- `己属性` - self.属性
- `对象.属性` - 对象.属性
- `类名.静态属性` - 类变量

## 异常处理

### 基本异常处理

```段言
尝试：
  定义文件等于打开参数"data.txt"。
  定义内容等于文件.读取。
  打印内容。
捕获 文件错误 为 错误：
  打印"无法读取文件："加错误.信息。
```

### 多异常捕获

```段言
尝试：
  定义数值等于转换整数参数"abc"。
捕获 数值错误 为 错误：
  打印"不是有效的数字"。
捕获 其他错误 为 错误：
  打印"发生未知错误"。
最终：
  打印"处理完成"。
```

### 抛出异常

```段言
《检查年龄》段(年龄)：
  如果年龄小于零：
    抛出 数值错误 参数"年龄不能为负数"。
  返回年龄。
```

## 编译器使用

### CLI 工具

**基本用法：**
```bash
# 解释执行
duan run file.duan

# 编译为 Python 文件
duan compile file.duan

# 指定输出文件
duan compile file.duan -o output.py

# 编译为 Windows EXE（需安装 PyInstaller）
duan compile file.duan -o output.exe

# 显示 Token 流
duan tokens file.duan

# 显示 AST
duan ast file.duan

# 选择后端（antlr / src / llvm）
duan run file.duan --backend antlr
duan compile file.duan --backend src

# 详细输出
duan run file.duan -v

# 启动 REPL
duan repl

# 查看版本
duan --version

# 查看帮助
duan --help
```

### 编译目标说明

| 命令 | 输出格式 | 说明 |
|------|---------|------|
| `duan compile file.duan` | `file.py` | 编译为 Python 源代码 |
| `duan compile file.duan -o file.exe` | `file.exe` | 编译为 Windows 可执行文件（需 PyInstaller） |
| `duan compile file.duan --backend llvm` | LLVM IR | 编译为 LLVM 中间表示 |

### Python API

```python
import sys
sys.path.insert(0, 'src')

from compiler import DuanCompiler
from code_generator_unified import UnifiedCodeGenerator

# 编译流程
compiler = DuanCompiler()
result = compiler.compile(source)

if result['success']:
    code_gen = UnifiedCodeGenerator()
    py_code = code_gen.generate(result['ast'])
    print(py_code)
else:
    print("编译错误:", result['errors'])
```

## 常见问题

### 1. 为什么我的代码编译失败？

**常见原因：**
- 缺少句号 `。`
- 关键字拼写错误
- 括号不匹配
- 字符串引号不匹配

**解决方法：**
使用 `--tokens` 查看词法分析结果：
```bash
python cli/duanc.py file.duan --tokens
```

### 2. 如何调试代码？

**方法1：** 使用 `--ast` 查看语法树
```bash
python cli/duanc.py file.duan --ast
```

**方法2：** 使用 `-v` 详细输出
```bash
python cli/duanc.py file.duan -v
```

### 3. 段言支持哪些关键字？

**核心关键字：**
- 定义：`定义`、`常量`、`类型`
- 条件：`如果`、`那么`、`否则`
- 循环：`遍历`、`当`、`跳出`、`跳过`
- 函数：`段`、`返回`
- 类：`类`、`继承`、`属性`、`构造`
- 模块：`模块`、`导入`、`导出`
- 异常：`尝试`、`捕获`、`抛出`、`最终`

### 4. 如何参与开发？

**步骤：**
1. Fork 项目仓库
2. 创建功能分支
3. 编写代码和测试
4. 提交 Pull Request

**开发指南：**
- 代码风格：遵循 PEP 8
- 测试要求：新增功能必须有测试
- 文档要求：新增功能必须更新文档
