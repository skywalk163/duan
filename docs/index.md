# 段言 (Duan) 文档

**段言**是一门基于中文的现代化编程语言，采用中文关键字，让编程更加直观易懂。

## 特性

- **中文关键字**：使用中文关键字如 `遍历`、`如果`、`那么` 等
- **双后端架构**：ANTLR 解析器 + 手写递归下降解析器
- **原生编译**：通过 LLVM IR 生成跨平台原生可执行文件
- **自举编译器**：段言语言使用自身实现编译器

## 安装

```bash
pip install duan
```

## 快速开始

```段言
打印 "你好，世界！"

段落 加一 接收 数：
    返回 数 加 1

如果 年龄 大于 18 那么：
    打印 "成年人"
```
