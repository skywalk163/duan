## 三、示例模块

### 3.1 math_utils.duan

```段言
# 数学工具模块

导出《平方》，《立方》。

《平方》段(数)：
  返回数乘数。

《立方》段(数)：
  返回数乘数乘数。
```

**编译为Python**:
```python
__all__ = ['平方', '立方']

def 平方(数):
    return (数 * 数)

def 立方(数):
    return ((数 * 数) * 数)
```

✅ **成功**

---

### 3.2 string_utils.duan

```段言
# 字符串工具模块

导出《连接》。

《连接》段(文本1, 文本2)：
  返回文本1加文本2。
```

**编译为Python**:
```python
__all__ = ['连接']

def 连接(文本1, 文本2):
    return (文本1 + 文本2)
```

✅ **成功**
