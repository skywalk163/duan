
# 段言记账程序

一个使用段言(DuanLang)编写的简单记账小程序。

## 功能特性

- ✅ 添加收入记录
- ✅ 添加支出记录
- ✅ 查看所有记录
- ✅ 收支统计（总收入、总支出、结余）
- ✅ 数据保存到JSON文件
- ✅ 自动加载历史数据

## 如何编译成可执行文件

### 前置条件

确保你已经安装了Python 3.8+，并且已安装段言编译器。

### 步骤一：安装段言编译器

在项目根目录执行：

```bash
pip install -e .
```

### 步骤二：编译记账程序

打开命令行，进入 `project` 目录：

```bash
cd project
```

使用段言编译器编译：

```bash
duan compile 记账.duan -o 记账.exe
```

### 步骤三：运行程序

编译成功后，直接运行生成的可执行文件：

```bash
记账.exe
```

或者双击 `记账.exe` 文件运行。

### 完整示例

```bash
# 进入项目目录
cd g:\dumategithub\duan\project

# 编译
duan compile 记账.duan -o 记账.exe

# 运行
记账.exe
```

## 使用说明

运行程序后，会显示菜单：

```
========== 记账程序 ==========
1. 添加收入
2. 添加支出
3. 查看记录
4. 收支统计
5. 保存数据
6. 退出程序
==============================
请输入选项(1-6):
```

### 添加收入/支出

1. 选择选项 `1`（收入）或 `2`（支出）
2. 输入金额（支持小数）
3. 输入备注

### 查看记录

选择选项 `3` 可以查看所有收支记录。

### 收支统计

选择选项 `4` 可以查看收支统计信息。

### 保存数据

选择选项 `5` 手动保存数据（退出时会自动保存）。

## 文件说明

| 文件 | 说明 |
|------|------|
| `记账.duan` | 段言源代码文件 |
| `记账数据.json` | 存储记账数据的JSON文件（自动生成） |
| `记账.exe` | 编译后的可执行文件（编译后生成） |

## 常见问题

### Q: 编译时提示找不到 duan 命令？

**A:** 确保已经安装了段言编译器：

```bash
pip install -e .
```

或者使用完整路径运行：

```bash
python -m cli.duan compile 记账.duan -o 记账.exe
```

### Q: 运行时提示缺少 DLL 文件？

**A:** 确保你的系统已安装必要的运行时库（如 Visual C++ Redistributable）。

### Q: 数据文件保存在哪里？

**A:** 数据保存在 `记账数据.json` 文件中，与程序同一目录。
