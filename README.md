# 段言（DuanLang）编程语言

**段言**是一门基于中文的现代化编程语言，采用中文关键字，让编程更加直观易懂。

## 特性

- **中文关键字**：使用中文关键字如`遍历`、`如果`、`那么`等
- **简洁语法**：基于缩进的简洁语法
- **多范式支持**：支持过程式和函数式编程
- **原生编译**：支持直接编译为跨平台原生可执行文件

## 安装

```bash
# 克隆仓库
git clone https://github.com/your-repo/duan.git
cd duan

# 安装依赖
pip install -r requirements.txt
```

## 快速开始

### Hello World

```duan
打印("你好，世界！")
```

### 变量和函数

```duan
《加一》段(数): 
    返回 数 加 1
。

打印(加一(5))  # 输出: 6
```

### 条件语句

```duan
如果 年龄 大于 18 那么:
    打印("成年人")
否则:
    打印("未成年人")
```

### 循环

```duan
遍历 项 在 列表:
    打印(项)
```

## 编译运行

### 方式1：Python解释器运行

```bash
python run.py project/你的程序.duan
```

### 方式2：编译为可执行文件

```bash
# 使用简单编译器
python simple_compiler.py project/你的程序.duan project/程序.py
python project/程序.py

# 或使用PyInstaller打包
pip install pyinstaller
pyinstaller --onefile project/程序.py -n 程序
```

### 方式3：原生编译器（实验性）

```bash
python -m src.compiler project/你的程序.duan -o 程序
./程序  # Linux/macOS
程序.exe  # Windows
```

## 语法参考

### 基本语法

| 段言 | 说明 |
|------|------|
| `定义 X 等于 Y` | 变量声明 |
| `《段名》段():` | 函数定义 |
| `《段名》()` | 函数调用 |
| `如果 条件 那么:` | 条件语句 |
| `遍历 X 在 列表:` | 遍历循环 |
| `当 条件:` | 当循环 |
| `返回 X` | 返回值 |
| `打印(X)` | 打印输出 |

### 数据类型

| 类型 | 说明 |
|------|------|
| `整数` | 整数类型 |
| `小数` | 浮点数类型 |
| `字符串` | 字符串类型 |
| `真假` | 布尔类型 |

### 运算符

| 段言 | 说明 |
|------|------|
| `加` | + |
| `减` | - |
| `乘` | * |
| `除` | / |
| `等于` | == |
| `大于` | > |
| `小于` | < |

## 示例程序

### 记账程序

完整的记账程序示例，展示段言的主要特性：

```duan
《主程序》段():
    记录列表 = []
    当 真:
        打印("请输入选项(1-6):")
        选项 = 到整数(读取行())
        
        如果 选项 等于 1 那么:
            添加收入(记录列表)
        否则 如果 选项 等于 2 那么:
            添加支出(记录列表)
        否则 如果 选项 等于 3 那么:
            查看记录(记录列表)
        否则 如果 选项 等于 4 那么:
            统计收支(记录列表)
        否则 如果 选项 等于 5 那么:
            保存数据(记录列表)
        否则 如果 选项 等于 6 那么:
            打印("再见！")
            返回
```
