# 段言（Duan）编程语言

**版本**: v1.0 MVP  
**状态**: 开发中

## 简介

段言是一门基于中文深层认知特性设计的编程语言，融合词句段篇层级、元数驱动解析、无空格分词等创新语法。

## 核心特性

- **双字关键字**：`定义` `等于` `如果` `那么` `返回` 等
- **符号双模式**：中文符号和英文符号完全等价
- **元数驱动解析**：动词声明参数数量，支持无括号调用
- **无空格分词**：类型切换自动分词
- **缩进定义块**：无需显式结束标记
- **主谓/谓宾语义**：`列表排序`原地修改 vs `排序列表`返回新对象


## 示例代码

```duan
# 定义变量
定义甲等于三。
定义乙等于五。
打印甲加乙。

# 定义段落
《快速排序》段(列表):
  如果列表长度小于二那么返回列表。
  
  定义基准等于列表之中间项。
  定义小列表等于列表过滤项小于基准。
  定义大列表等于列表过滤项大于基准。
  
  返回《快速排序》(小列表) -> 等列表 -> 《快速排序》(大列表)。

# 主谓/谓宾语义
定义甲等于列5 3 1 4 2。
甲排序。              # 原地修改
定义乙等于排序甲。    # 返回新对象
```
